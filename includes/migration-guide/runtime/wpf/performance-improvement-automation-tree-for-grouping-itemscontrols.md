### <a name="performance-improvement-in-automation-tree-for-grouping-itemscontrols"></a>Mejora del rendimiento en el árbol de automatización para agrupar instancias ItemsControls

#### <a name="details"></a>Detalles

Mejora el rendimiento de recompilar el árbol de automatización de <xref:System.Windows.Controls.ItemsControl>, como ListBox o DataGrid, en el que está habilitada la agrupación.

|:--------|:------------|
