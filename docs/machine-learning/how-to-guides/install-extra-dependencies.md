### <a name="linux"></a><span data-ttu-id="010ae-113">Linux</span><span class="sxs-lookup"><span data-stu-id="010ae-113">Linux</span></span>

1. <span data-ttu-id="010ae-114">Instalar la clave GPG para el repositorio</span><span class="sxs-lookup"><span data-stu-id="010ae-114">Install the GPG key for the repository</span></span>

    ```bash
    sudo bash
    # <type your user password when prompted.  this will put you in a root shell>
    # cd to /tmp where this shell has write permission
    cd /tmp
    # now get the key:
    wget https://apt.repos.intel.com/intel-gpg-keys/GPG-PUB-KEY-INTEL-SW-PRODUCTS-2019.PUB
    # now install that key
    apt-key add GPG-PUB-KEY-INTEL-SW-PRODUCTS-2019.PUB
    # now remove the public key file exit the root shell
    rm GPG-PUB-KEY-INTEL-SW-PRODUCTS-2019.PUB
    exit
    ```

2. <span data-ttu-id="010ae-115">Agregar el repositorio APT para MKL</span><span class="sxs-lookup"><span data-stu-id="010ae-115">Add the APT Repository for MKL</span></span>

    ```bash
    sudo sh -c 'echo deb https://apt.repos.intel.com/mkl all main > /etc/apt/sources.list.d/intel-mkl.list'
    ```

3. <span data-ttu-id="010ae-116">Actualizar paquetes</span><span class="sxs-lookup"><span data-stu-id="010ae-116">Update packages</span></span>

    ```bash
    sudo apt-get update
    ```

4. <span data-ttu-id="010ae-117">Instalar MKL</span><span class="sxs-lookup"><span data-stu-id="010ae-117">Install MKL</span></span>

    ```bash
    sudo apt-get install <COMPONENT>-<VERSION>.<UPDATE>-<BUILD_NUMBER>
    ```

    <span data-ttu-id="010ae-118">Por ejemplo:</span><span class="sxs-lookup"><span data-stu-id="010ae-118">For example:</span></span>

    ```bash
    sudo apt-get install intel-mkl-64bit-2020.0-088
    ```

    <span data-ttu-id="010ae-119">Determinar la ubicación de `libiomp.so`</span><span class="sxs-lookup"><span data-stu-id="010ae-119">Determine the location of `libiomp.so`</span></span>

    ```bash
    find /opt -name "libiomp5.so"
    ```

    <span data-ttu-id="010ae-120">Por ejemplo:</span><span class="sxs-lookup"><span data-stu-id="010ae-120">For example:</span></span>

    ```output
    /opt/intel/compilers_and_libraries_2020.0.166/linux/compiler/lib/intel64_lin/libiomp5.so
    ```

5. <span data-ttu-id="010ae-121">Agregue esta ubicación a la ruta de acceso de la biblioteca de carga:</span><span class="sxs-lookup"><span data-stu-id="010ae-121">Add this location to the load library path:</span></span>

    ```bash
    sudo ldconfig /opt/intel/compilers_and_libraries_2020.0.166/linux/compiler/lib/intel64_lin
    ```
