## <a name="example"></a><span data-ttu-id="dfbfb-126">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="dfbfb-126">Example</span></span>  
 <span data-ttu-id="dfbfb-127">En el siguiente ejemplo se muestra el manifiesto predeterminado que el compilador de Visual Basic inserta en un archivo PE.</span><span class="sxs-lookup"><span data-stu-id="dfbfb-127">The following example shows the default manifest that the Visual Basic compiler inserts into a PE.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="dfbfb-128">El compilador inserta un nombre de aplicación estándar MyApplication.app en el archivo XML de manifiesto.</span><span class="sxs-lookup"><span data-stu-id="dfbfb-128">The compiler inserts a standard application name MyApplication.app into the manifest XML.</span></span> <span data-ttu-id="dfbfb-129">Se trata de una solución alternativa para permitir que las aplicaciones se ejecuten en Windows Server 2003 Service Pack 3.</span><span class="sxs-lookup"><span data-stu-id="dfbfb-129">This is a workaround to enable applications to run on Windows Server 2003 Service Pack 3.</span></span>

    <security>  
      <requestedPrivileges xmlns="urn:schemas-microsoft-com:asm.v3">

      </requestedPrivileges>  
    </security>
