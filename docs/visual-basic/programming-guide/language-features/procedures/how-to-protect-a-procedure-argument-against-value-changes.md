<span data-ttu-id="74606-103">Si un procedimiento declara un parámetro como [ByRef](../../../language-reference/modifiers/byref.md), Visual Basic proporciona al código de procedimiento una referencia directa al elemento de programación subyacente al argumento en el código de llamada.</span><span class="sxs-lookup"><span data-stu-id="74606-103">If a procedure declares a parameter as [ByRef](../../../language-reference/modifiers/byref.md), Visual Basic gives the procedure code a direct reference to the programming element underlying the argument in the calling code.</span></span> <span data-ttu-id="74606-104">Esto permite al procedimiento cambiar el valor subyacente del argumento en el código de llamada.</span><span class="sxs-lookup"><span data-stu-id="74606-104">This permits the procedure to change the value underlying the argument in the calling code.</span></span> <span data-ttu-id="74606-105">En algunos casos, es posible que el código de llamada desee proteger contra este tipo de cambio.</span><span class="sxs-lookup"><span data-stu-id="74606-105">In some cases the calling code might want to protect against such a change.</span></span>  
  
 <span data-ttu-id="74606-106">Siempre puede proteger un argumento del cambio declarando el parámetro correspondiente [ByVal](../../../language-reference/modifiers/byval.md) en el procedimiento.</span><span class="sxs-lookup"><span data-stu-id="74606-106">You can always protect an argument from change by declaring the corresponding parameter [ByVal](../../../language-reference/modifiers/byval.md) in the procedure.</span></span> <span data-ttu-id="74606-107">Si desea poder cambiar un argumento determinado en algunos casos, pero no en otros, puede declararlo `ByRef` y permitir que el código de llamada determine el mecanismo de paso en cada llamada.</span><span class="sxs-lookup"><span data-stu-id="74606-107">If you want to be able to change a given argument in some cases but not others, you can declare it `ByRef` and let the calling code determine the passing mechanism in each call.</span></span> <span data-ttu-id="74606-108">Para ello, incluye el argumento correspondiente entre paréntesis para pasarlo por valor o no lo incluye entre paréntesis para pasarlo por referencia.</span><span class="sxs-lookup"><span data-stu-id="74606-108">It does this by enclosing the corresponding argument in parentheses to pass it by value, or not enclosing it in parentheses to pass it by reference.</span></span> <span data-ttu-id="74606-109">Para obtener más información, vea [Cómo: forzar un argumento para que se pase por valor](./how-to-force-an-argument-to-be-passed-by-value.md).</span><span class="sxs-lookup"><span data-stu-id="74606-109">For more information, see [How to: Force an Argument to Be Passed by Value](./how-to-force-an-argument-to-be-passed-by-value.md).</span></span>
