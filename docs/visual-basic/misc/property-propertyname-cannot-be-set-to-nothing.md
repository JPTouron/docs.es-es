# <a name="property-propertyname-cannot-be-set-to-nothing"></a>La propiedad \<propertyname> no se puede establecer como Nothing

La propiedad requiere un valor distinto de `Nothing`. Esto puede ser el resultado del cálculo incorrecto del valor.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Facilite un valor válido.
