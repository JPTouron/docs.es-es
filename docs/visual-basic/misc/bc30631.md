# <a name="get-statement-must-end-with-a-matching-end-get"></a>La instrucción 'Get' debe terminar con la instrucción 'End Get' correspondiente.
Los procedimientos de propiedad`Get` deben terminar con una instrucción `End Get` .  
  
 **Identificador de error:** BC30631  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de que el procedimiento de propiedad `Get` termine con una construcción `End Get` .
