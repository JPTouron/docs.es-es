# <a name="method-methodname1-must-be-declared-private-in-order-to-implement-partial-method-methodname2"></a><span data-ttu-id="5b940-102">El método ' \<methodname1> ' debe declararse como ' Private ' para poder implementar el método parcial ' \<methodname2> '</span><span class="sxs-lookup"><span data-stu-id="5b940-102">Method '\<methodname1>' must be declared 'Private' in order to implement partial method '\<methodname2>'</span></span>

<span data-ttu-id="5b940-103">La implementación de un método parcial debe declararse `Private`.</span><span class="sxs-lookup"><span data-stu-id="5b940-103">The implementation of a partial method must be declared `Private`.</span></span> <span data-ttu-id="5b940-104">Por ejemplo, el código siguiente causa este error.</span><span class="sxs-lookup"><span data-stu-id="5b940-104">For example, the following code causes this error.</span></span>  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,
    ' causes this error.
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 <span data-ttu-id="5b940-105">**Identificador de error:** BC31441</span><span class="sxs-lookup"><span data-stu-id="5b940-105">**Error ID:** BC31441</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5b940-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="5b940-106">To correct this error</span></span>  
  
1. <span data-ttu-id="5b940-107">Use el modificador de acceso `Private` en la implementación del método parcial, tal como se muestra en el ejemplo siguiente.</span><span class="sxs-lookup"><span data-stu-id="5b940-107">Use the access modifier `Private` in the implementation of the partial method, as shown in the following example.</span></span>  
  
```vb

    MsgBox("Value was changed to " & Me.Quantity)  
End Sub  
```
