# <a name="option-optionname-requires-argument"></a>La opción '\<optionname>' requiere \<argument>

Se ha omitido un argumento requerido.  
  
 **Id. de error:** BC2006  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Facilite el argumento que falta.
