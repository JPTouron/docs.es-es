# <a name="unable-to-create-a-net-runtime-interface-error"></a>No se puede crear una interfaz de .NET Runtime: \<error>

Hubo un error interno del compilador al crear una interfaz o un error de llamada a un método en una interfaz.  
  
 **Identificador de error:** BC31024  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Guarde el trabajo y reinicie Visual Studio.  
  
2. Si el error se repite, vuelva a instalar Visual Studio.  
  
3. Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
