# <a name="type-cannot-be-inherited-more-than-once"></a>' \<type> ' no se puede heredar más de una vez
Duplicó una instrucción `Inherits` .  
  
 **Identificador de error:** BC30584  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la instrucción `Inherits` redundante.
