# <a name="file-filename-cannot-be-deleted-because-it-is-open"></a>No se puede eliminar el archivo '\<filename>' porque está abierto

Se ha intentado una eliminación en un archivo abierto.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Cierre el archivo antes de eliminarlo.
