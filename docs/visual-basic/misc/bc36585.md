# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>' \<elementname> ' no es un parámetro del método de extensión ' \<methodname> ' definido en ' \<typename> '
Una llamada a procedimiento ofrece un argumento adicional que no está definido por el método de extensión.  
  
 **Id. de error:** BC36585  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el argumento superfluo de la lista de argumentos.
