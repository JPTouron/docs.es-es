# <a name="cannot-find-net-framework-directory-error"></a>No se encuentra el directorio .NET Framework: \<error>

No se pudo encontrar el directorio donde se almacenan los ensamblados de .NET Framework.  
  
 **Id. de error:** BC31508  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Vuelva a compilar el programa para ver si el error se repite.  
  
2. Si el error continúa, guarde el trabajo y reinicie Visual Studio.  
  
3. Si el error se repite, reinstale Visual Basic.  
  
4. Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
