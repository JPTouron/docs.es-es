# <a name="classes-cannot-be-declared-specifier"></a>Las clases no se pueden declarar como ' \<specifier> '

Ha intentado declarar una `Class` o un `Module` con un especificador no válido.  
  
 **Identificador de error:** BC30461  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el especificador.
