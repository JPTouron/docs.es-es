# <a name="partial-method-methodname-cannot-use-the-implements-keyword"></a>El método parcial '\<methodname>' no puede usar la palabra clave 'Implements'

Una definición de método parcial no puede implementar una interfaz o un miembro de interfaz.  
  
 **Identificador de error:** BC31439  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Elimine la instrucción `Implements` de la definición de método parcial.
