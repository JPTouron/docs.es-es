# <a name="instance-members-and-me-cannot-be-used-in-a-query-expression"></a><span data-ttu-id="e4fd0-102">No se pueden usar miembros de instancia ni 'Me' en una expresión de consulta.</span><span class="sxs-lookup"><span data-stu-id="e4fd0-102">Instance members and 'Me' cannot be used in a query expression</span></span>

<span data-ttu-id="e4fd0-103">Una consulta LINQ en una `Structure` incluye una referencia a `Me` o a un miembro de instancia de la estructura.</span><span class="sxs-lookup"><span data-stu-id="e4fd0-103">A LINQ query in a `Structure` includes a reference to `Me` or to an instance member of the structure.</span></span> <span data-ttu-id="e4fd0-104">Las referencias a `Me` o a los miembros de instancia no se permiten en expresiones de consulta dentro de un `Structure`.</span><span class="sxs-lookup"><span data-stu-id="e4fd0-104">References to `Me` or instance members are not allowed in query expressions within a `Structure`.</span></span>  
  
 <span data-ttu-id="e4fd0-105">**Identificador de error:** BC36535</span><span class="sxs-lookup"><span data-stu-id="e4fd0-105">**Error ID:** BC36535</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e4fd0-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="e4fd0-106">To correct this error</span></span>  
  
1. <span data-ttu-id="e4fd0-107">Cree una copia del miembro de instancia o el valor devuelto por la referencia a `Me` y use la copia en la expresión de consulta, como se muestra en el ejemplo siguiente.</span><span class="sxs-lookup"><span data-stu-id="e4fd0-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb  
Structure SampleStructure  
    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv  
    End Sub  
End Structure  
```
