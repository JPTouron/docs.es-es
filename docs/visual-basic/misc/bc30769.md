# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-end-statement"></a>La versión de destino de .NET Compact Framework no admite la instrucción 'End'.
La versión de .NET Compact Framework con la que está trabajando no admite la declaración `End` .  
  
 **Identificador de error:** BC30769  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la instrucción.
