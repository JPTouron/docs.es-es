# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="48dc9-102">El miembro de tipo anónimo o la propiedad '\<propertyname>' están ya declarados</span><span class="sxs-lookup"><span data-stu-id="48dc9-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="48dc9-103">Un nombre de propiedad puede establecerse solo una vez en la declaración de un tipo anónimo.</span><span class="sxs-lookup"><span data-stu-id="48dc9-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="48dc9-104">Por ejemplo, las siguientes declaraciones no son válidas:</span><span class="sxs-lookup"><span data-stu-id="48dc9-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="48dc9-105">**Identificador de error:** BC36547</span><span class="sxs-lookup"><span data-stu-id="48dc9-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="48dc9-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="48dc9-106">To correct this error</span></span>  
  
- <span data-ttu-id="48dc9-107">Elija otro nombre para una de las propiedades.</span><span class="sxs-lookup"><span data-stu-id="48dc9-107">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.  
Dim anonType3 = New With {.Name = "Debra Garcia", .Label = .Name & ", President"}  
```  
  
- <span data-ttu-id="48dc9-108">Proporcione nombres nuevos para las variables o los nombres de propiedad de las que se están infiriendo nombres y valores.</span><span class="sxs-lookup"><span data-stu-id="48dc9-108">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```
