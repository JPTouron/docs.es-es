# <a name="methods-in-a-module-cannot-be-declared-specifier"></a>Los métodos de un módulo no se pueden declarar como ' \<specifier> '
Se ha usado un especificador que no es válido en un método dentro de una instrucción `Module` . Los módulos no admiten la creación de instancias ni la herencia y tampoco pueden implementar interfaces.  
  
 **Identificador de error:** BC30433  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el especificador.
