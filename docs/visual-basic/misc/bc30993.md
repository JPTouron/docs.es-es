# <a name="property-propertyname-cannot-be-initialized-in-an-object-initializer-expression-because-all-accessible-overloads-require-arguments"></a><span data-ttu-id="5d656-102">No se puede inicializar la propiedad '\<propertyname>' en una expresión de inicializador de objeto porque todas las sobrecargas accesibles requieren argumentos</span><span class="sxs-lookup"><span data-stu-id="5d656-102">Property '\<propertyname>' cannot be initialized in an object initializer expression because all accessible overloads require arguments</span></span>
<span data-ttu-id="5d656-103">Los miembros que se inicializan en una lista de inicializadores de objeto deben ser campos o propiedades.</span><span class="sxs-lookup"><span data-stu-id="5d656-103">The members initialized in an object initializer list must be either fields or properties.</span></span> <span data-ttu-id="5d656-104">Además, las propiedades de una lista de inicializador no pueden tener parámetros.</span><span class="sxs-lookup"><span data-stu-id="5d656-104">Additionally, properties in an initializer list cannot have parameters.</span></span> <span data-ttu-id="5d656-105">La propiedad que causa el error está sobrecargada y cada una de sus versiones requiere argumentos.</span><span class="sxs-lookup"><span data-stu-id="5d656-105">The property causing this error is overloaded, and each of its versions requires arguments.</span></span> <span data-ttu-id="5d656-106">Por lo tanto, la propiedad no se puede inicializar en una lista de inicializadores de objeto.</span><span class="sxs-lookup"><span data-stu-id="5d656-106">Therefore, the property cannot be initialized in an object initializer list.</span></span>  
  
 <span data-ttu-id="5d656-107">**Identificador de error:** BC30993</span><span class="sxs-lookup"><span data-stu-id="5d656-107">**Error ID:** BC30993</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5d656-108">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="5d656-108">To correct this error</span></span>  
  
- <span data-ttu-id="5d656-109">Quite la propiedad que requiere argumentos de la lista de inicializadores.</span><span class="sxs-lookup"><span data-stu-id="5d656-109">Remove the property that requires arguments from the initializer list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5d656-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="5d656-110">Example</span></span>  
 <span data-ttu-id="5d656-111">La siguiente clase contiene tres definiciones de propiedad: una para `TotalItems` y dos para `Item`, que está sobrecargado.</span><span class="sxs-lookup"><span data-stu-id="5d656-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```vb  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="5d656-112">La propiedad `TotalItems` no requiere argumentos y se puede inicializar en una lista de inicialización de objetos, tal como se muestra en la siguiente declaración.</span><span class="sxs-lookup"><span data-stu-id="5d656-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="5d656-113">La propiedad `Item` está sobrecargada y cada sobrecarga requiere un argumento.</span><span class="sxs-lookup"><span data-stu-id="5d656-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="5d656-114">Por lo tanto, `Item` no puede aparecer en una lista de inicializadores de objeto.</span><span class="sxs-lookup"><span data-stu-id="5d656-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```vb  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="5d656-115">Para evitar este error, inicialice la propiedad `Item` fuera del inicializador de objeto.</span><span class="sxs-lookup"><span data-stu-id="5d656-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```vb  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
