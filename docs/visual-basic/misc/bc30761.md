# <a name="project-projectname-cannot-reference-project-projectname-because-projectname-directly-or-indirectly-references-projectname"></a>El proyecto ' \<projectname> ' no puede hacer referencia al proyecto ' \<projectname> ' porque ' \<projectname> ' hace referencia directa o indirectamente a ' \<projectname> '

Un proyecto incluye una referencia a un segundo proyecto. A su vez, el segundo proyecto incluye una referencia al primer proyecto.  
  
 **Identificador de error:** BC30761  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite una de las referencias.
