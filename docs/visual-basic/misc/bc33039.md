# <a name="return-and-parameter-types-of-operator-must-be-typename-to-be-used-in-a-for-statement"></a><span data-ttu-id="2f1bf-102">Los tipos de valor devuelto y de parámetro de '\<operator>' deben ser '\<typename>' para usarse en una expresión 'For'</span><span class="sxs-lookup"><span data-stu-id="2f1bf-102">Return and parameter types of '\<operator>' must be '\<typename>' to be used in a 'For' statement</span></span>

<span data-ttu-id="2f1bf-103">Un bucle `For` especifica una variable de contador de un tipo que no define el operador `+` o `-` con parámetros y el valor devuelto de su propio tipo.</span><span class="sxs-lookup"><span data-stu-id="2f1bf-103">A `For` loop specifies a counter variable of a type that does not define the `+` or `-` operator with parameters and return value of its own type.</span></span>  
  
 <span data-ttu-id="2f1bf-104">La variable de contador debe ser de un tipo que admita los operadores de suma (`+`) y resta (`-`) que funcionen plenamente en su tipo contenedor.</span><span class="sxs-lookup"><span data-stu-id="2f1bf-104">The counter variable must be of a type that supports addition (`+`) and subtraction (`-`) operators that operate completely on their containing type.</span></span> <span data-ttu-id="2f1bf-105">Esto significa que ambos operandos y el valor devuelto deben ser del tipo de la variable de contador.</span><span class="sxs-lookup"><span data-stu-id="2f1bf-105">This means both of the operands and the return value must be of the type of the counter variable.</span></span>  
  
 <span data-ttu-id="2f1bf-106">Si usa un tipo de datos numérico para la variable de contador, se admiten los operadores `+` y `-` en el tipo contenedor.</span><span class="sxs-lookup"><span data-stu-id="2f1bf-106">If you use a numeric data type for the counter variable, the `+` and `-` operators are supported on the containing type.</span></span> <span data-ttu-id="2f1bf-107">Si usa una clase o una estructura definida por el usuario, debe definir ambos operadores con los operandos y el valor devuelto del tipo de dicha clase o estructura.</span><span class="sxs-lookup"><span data-stu-id="2f1bf-107">If you use a user-defined class or structure, you must define both operators with operands and return value of the type of your class or structure.</span></span>  
  
 <span data-ttu-id="2f1bf-108">**Identificador de error:** BC33039</span><span class="sxs-lookup"><span data-stu-id="2f1bf-108">**Error ID:** BC33039</span></span>
