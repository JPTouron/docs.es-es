# <a name="operator-operatorname-is-not-defined-for-types-type1-and-type2"></a>El operador ' \<operatorname> ' no está definido para los tipos ' \<type1> ' y ' \<type2> '
Ha intentado utilizar un operador binario para realizar una operación que no es válida para los dos elementos de código que devuelven valores implicados.  
  
 **Id. de error:** BC30452  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Compruebe los dos elementos y asegúrese de que sean compatibles.
