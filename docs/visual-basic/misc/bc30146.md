# <a name="unable-to-sign-assembly-error-message"></a>No se puede firmar el ensamblado: \<error message>

El compilador Visual Basic llama a Assembly Linker (Al.exe, también conocido como ALink) para generar un ensamblado con un manifiesto. El enlazador ha informado de un error al firmar el ensamblado después de su creación. Puede deberse a una clave no válida.  
  
 **Identificador de error:** BC30146  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Examine el mensaje de error citado y consulte el tema  [Al.exe (Assembly Linker)](../../framework/tools/al-exe-assembly-linker.md) para obtener una explicación más detallada y consejos.  
  
2. Si el error persiste, reúna información sobre las circunstancias y notifíquelo a los Servicios de soporte técnico de Microsoft.
