# <a name="array-exceeds-the-limit-of-32-dimensions"></a>La matriz supera el límite de 32 dimensiones
Las matrices no pueden tener más de 32 dimensiones.  
  
 **Identificador de error:** BC30052  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Reduzca el número de dimensiones en la declaración de la matriz.
