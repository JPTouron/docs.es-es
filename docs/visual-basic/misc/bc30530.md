# <a name="parameter-cannot-have-the-same-name-as-its-defining-function"></a>El parámetro no puede tener el mismo nombre que la función que lo define.

Ha usado el mismo nombre en un parámetro y su función de definición.  
  
 **Identificador de error:** BC30530  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Compruebe los nombres de parámetro dentro de la función y cambie el nombre del parámetro en conflicto.
