# <a name="mustoverride-cannot-be-specified-on-procedurename-because-it-is-in-a-partial-type-that-is-declared-notinheritable-in-another-partial-definition"></a><span data-ttu-id="8e5e5-102">'MustOverride' no se puede especificar en '\<procedurename>' porque se encuentra en un tipo parcial que se declaró como 'NotInheritable' en otra definición parcial</span><span class="sxs-lookup"><span data-stu-id="8e5e5-102">'MustOverride' cannot be specified on '\<procedurename>' because it is in a partial type that is declared 'NotInheritable' in another partial definition</span></span>
<span data-ttu-id="8e5e5-103">Se declaró un procedimiento o una propiedad como `MustOverride` dentro de una clase definida en varias declaraciones parciales, pero una de estas definiciones parciales especifica `NotInheritable` para la clase.</span><span class="sxs-lookup"><span data-stu-id="8e5e5-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="8e5e5-104">Cuando se divide la definición de una clase entre varias declaraciones parciales, el compilador trata a la clase tipo como la unión de todas sus declaraciones parciales.</span><span class="sxs-lookup"><span data-stu-id="8e5e5-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="8e5e5-105">Esto se aplica no solo a los miembros, sino también a la implementación, la herencia y el nivel de acceso.</span><span class="sxs-lookup"><span data-stu-id="8e5e5-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="8e5e5-106">Para invalidar un procedimiento o una propiedad, una clase debe heredar dicho procedimiento o propiedad de una clase base.</span><span class="sxs-lookup"><span data-stu-id="8e5e5-106">To override a procedure or property, a class must inherit it from a base class.</span></span> <span data-ttu-id="8e5e5-107">Por lo tanto, para especificar `MustOverride` para un procedimiento o una propiedad en una clase base, se debe especificar `MustInherit` para la clase.</span><span class="sxs-lookup"><span data-stu-id="8e5e5-107">Therefore, to specify `MustOverride` for a procedure or property in a base class, you must specify `MustInherit` for the class.</span></span> <span data-ttu-id="8e5e5-108">Dado que son mutuamente contradictorios, no es posible especificar ambos, `MustInherit` y `NotInheritable` , para la misma clase.</span><span class="sxs-lookup"><span data-stu-id="8e5e5-108">Because they are mutually contradictory, you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>  
  
 <span data-ttu-id="8e5e5-109">**Identificador de error:** BC30927</span><span class="sxs-lookup"><span data-stu-id="8e5e5-109">**Error ID:** BC30927</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8e5e5-110">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="8e5e5-110">To correct this error</span></span>
