# <a name="the-keyword-keyword-is-used-to-overload-inherited-members-do-not-use-the-keyword-keyword-when-overloading-sub-new"></a>La palabra clave '\<keyword>' se usa para sobrecargar los miembros heredados; no use la palabra clave '\<keyword>' cuando sobrecargue un procedimiento 'Sub New'
Se ha declarado un constructor con la palabra clave `Overloads` .  
  
 Visual Basic no admite la herencia ni la sobrecarga de constructores.  
  
 **Identificador de error:** BC32040  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la palabra clave `Overloads` de todas las declaraciones de constructor.
