# <a name="namespace-or-type-name-has-already-been-imported"></a>Ya se ha importado el espacio de nombres o tipo '\<name>'
Se intentó importar un espacio de nombres más de una vez.  
  
 **Identificador de error:** BC31051  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite las instrucciones `Imports` redundantes.
