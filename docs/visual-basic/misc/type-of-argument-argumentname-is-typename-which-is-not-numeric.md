# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>El tipo de argumento '\<argumentname>' es '\<typename>', que no es numérico

Un argumento no numérico se produce en una situación en la que se espera un argumento numérico.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asegúrese de que el argumento sea numérico.
