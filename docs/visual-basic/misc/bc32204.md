# <a name="event-name-length-cannot-exceed-1011-characters"></a>La longitud del nombre del evento no puede superar los 1011 caracteres.

El nombre o identificador de cada evento declarado con una instrucción `Event` está limitado a 1011 caracteres.  
  
 **Identificador de error:** BC32204  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Reduzca la longitud del identificador.
