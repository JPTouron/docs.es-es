# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-if-operator"></a><span data-ttu-id="d815c-102">No se puede inferir un tipo común para los operandos primero y segundo del operador 'If' binario</span><span class="sxs-lookup"><span data-stu-id="d815c-102">Cannot infer a common type for the first and second operands of the binary 'If' operator</span></span>

<span data-ttu-id="d815c-103">No se puede inferir un tipo común para los operandos primero y segundo del operador 'If' binario.</span><span class="sxs-lookup"><span data-stu-id="d815c-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="d815c-104">Uno debe tener una conversión de ampliación al otro.</span><span class="sxs-lookup"><span data-stu-id="d815c-104">One must have a widening conversion to the other.</span></span>

<span data-ttu-id="d815c-105">El operador `If` binario requiere que exista una conversión de ampliación entre uno de los argumentos y el otro.</span><span class="sxs-lookup"><span data-stu-id="d815c-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="d815c-106">Por ejemplo, como no existe una conversión de ampliación en ninguna dirección entre `Integer` y `String`, el código siguiente provoca este error.</span><span class="sxs-lookup"><span data-stu-id="d815c-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb
Dim first? As Integer
Dim second As String = "First is Nothing"
'' Not valid.
' Console.WriteLine(If(first, second))
```

<span data-ttu-id="d815c-107">**Identificador de error:** BC33110</span><span class="sxs-lookup"><span data-stu-id="d815c-107">**Error ID:** BC33110</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="d815c-108">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="d815c-108">To correct this error</span></span>

  ```vb

  ```

- <span data-ttu-id="d815c-110">Vuelva a escribir el código mediante una construcción condicional diferente.</span><span class="sxs-lookup"><span data-stu-id="d815c-110">Rewrite the code by using a different conditional construction.</span></span>

  ```vb
  If first IsNot Nothing Then
      Console.WriteLine(first)
  Else
      Console.WriteLine(second)
  End If
  ```
