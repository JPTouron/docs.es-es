# <a name="withevents-variable-does-not-raise-any-events"></a>La variable 'WithEvents' no activa eventos

La variable de objeto declarada con la palabra clave `WithEvents` no hace referencia a una instancia de clase que pueda activar eventos.  
  
 **Id. de error:** BC30591  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Asegúrese de que la variable de objeto hace referencia a una instancia de clase que pueda activar eventos.
