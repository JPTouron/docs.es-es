# <a name="cannot-convert-argument-argumentname-of-type-type1-to-type-type2"></a>No se puede convertir el argumento ' \<argumentname> ' de tipo ' \<type1> ' al tipo ' \<type2> '

Una operación de conversión intenta convertir un argumento a un tipo incompatible.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Asegúrese de que el tipo al que está intentando convertir el argumento sea válido.
