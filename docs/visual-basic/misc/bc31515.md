# <a name="project-projectname-makes-an-indirect-reference-to-assembly-assemblyname-which-contains-classname"></a>El proyecto ' \<projectname> ' hace una referencia indirecta al ensamblado ' \<assemblyname> ' que contiene ' \<classname> '

El proyecto ' \<projectname> ' hace una referencia indirecta al ensamblado ' \<assemblyname> ' que contiene ' \<classname> '. Agregue una referencia a ' \<assemblyname> ' al proyecto.  
  
 Este proyecto contiene miembros que dependen de un ensamblado que no está incluido en las referencias del proyecto.  
  
 **Identificador de error:** BC31515  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Haga clic con el botón derecho en **Referencias** en el **Explorador de soluciones** y, después, haga clic en **Agregar referencia**para agregar una referencia al ensamblado.
