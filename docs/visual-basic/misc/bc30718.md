# <a name="select-statements-are-not-valid-in-the-immediate-window"></a>Las instrucciones 'Select' no son válidas en la ventana Inmediato

Las instrucciones`Select` solo pueden usarse en el código fuente.  
  
 **Identificador de error:** BC30718  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite las instrucciones `Select` del código de depuración.
