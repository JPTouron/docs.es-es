# <a name="extension-method-methodname-has-type-constraints-that-can-never-be-satisfied"></a><span data-ttu-id="723a4-102">El método de extensión '\<methodname>' tiene restricciones de tipo que no se pueden cumplir nunca</span><span class="sxs-lookup"><span data-stu-id="723a4-102">Extension method '\<methodname>' has type constraints that can never be satisfied</span></span>

<span data-ttu-id="723a4-103">Los parámetros de tipo de este método interactúan de forma que les impide que alguna vez se cumplan.</span><span class="sxs-lookup"><span data-stu-id="723a4-103">The type parameters of this method interact in a way that prevents them from ever being satisfied.</span></span> <span data-ttu-id="723a4-104">El siguiente método de extensión es un ejemplo.</span><span class="sxs-lookup"><span data-stu-id="723a4-104">The following extension method is an example.</span></span>  
  
```vb  
'' Not valid.  
'<Extension()> _  
'Sub extensionExample(Of T As U, U)(ByVal para1 As T, ByVal para2 As U)  
'End Sub  
```  
  
 <span data-ttu-id="723a4-105">Dado que el método es un método de extensión, el compilador debe ser capaz de determinar el tipo o tipos de datos que el método extiende basándose únicamente en el primer parámetro de la declaración de método, `para1`, y el argumento enviado para ese parámetro.</span><span class="sxs-lookup"><span data-stu-id="723a4-105">Because the method is an extension method, the compiler must be able to determine the data type or types that the method extends based only on the first parameter in the method declaration, `para1`, and the argument sent in for that parameter.</span></span> <span data-ttu-id="723a4-106">Cuando el primer parámetro hace referencia a parámetros de tipo genérico, `para1 as T`, las restricciones en los parámetros genéricos restringen el conjunto de tipos al que se aplica el método.</span><span class="sxs-lookup"><span data-stu-id="723a4-106">When the first parameter refers to generic type parameters, `para1 as T`, the constraints on the generic parameters restrict the set of types to which the method applies.</span></span>  
  
 <span data-ttu-id="723a4-107">La aplicabilidad de un método de extensión se determina a partir del argumento proporcionado para el primer parámetro, que es `arg1` en el código siguiente.</span><span class="sxs-lookup"><span data-stu-id="723a4-107">Applicability of an extension method is determined from the argument provided for the first parameter, which is `arg1` in the following code.</span></span>  
  
 `'' Not valid.`  
  
 `'arg1.extensionExample(arg2)`  
  
 <span data-ttu-id="723a4-108">Debe ser posible comprobar las restricciones en todos los parámetros de tipo genérico que hace referencia el primer parámetro, `para1`, examinando solo el primer argumento, `arg1`.</span><span class="sxs-lookup"><span data-stu-id="723a4-108">It must be possible to verify the constraints on all generic type parameters referred to by the first parameter, `para1`, by looking at only the first argument, `arg1`.</span></span> <span data-ttu-id="723a4-109">En `extensionExample`, no se puede determinar el conjunto de tipos que se extiende a partir del primer parámetro solo.</span><span class="sxs-lookup"><span data-stu-id="723a4-109">In `extensionExample`, the set of types that is being extended cannot be determined from the first parameter alone.</span></span> <span data-ttu-id="723a4-110">Parámetro de tipo `T` está restringido por el parámetro de tipo `U`, al que `para1` no hace referencia y que no se puede inferir de `arg1`.</span><span class="sxs-lookup"><span data-stu-id="723a4-110">Type parameter `T` is constrained by type parameter `U`, which is not referenced by `para1` and cannot be inferred from `arg1`.</span></span> <span data-ttu-id="723a4-111">Por lo tanto, no se puede comprobar la aplicabilidad del método a cualquier tipo posible y nunca se puede llamar al método.</span><span class="sxs-lookup"><span data-stu-id="723a4-111">Therefore, the applicability of the method to any possible type cannot be verified, and the method can never be called.</span></span>  
  
 <span data-ttu-id="723a4-112">**Identificador de error:** BC36561</span><span class="sxs-lookup"><span data-stu-id="723a4-112">**Error ID:** BC36561</span></span>
