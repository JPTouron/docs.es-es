# <a name="xml-comment-tag-tag-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>La etiqueta de comentario XML '\<tag>' aparece con atributos idénticos más de una vez en el mismo bloque de comentario XML
La etiqueta de comentario XML \<tag> aparece con atributos idénticos más de una vez en el mismo bloque de comentario XML. Se omitirá el comentario XML.  
  
 El comentario XML contiene más de una etiqueta con atributos idénticos.  
  
 **Id. de error:** BC42305  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la etiqueta duplicada.
