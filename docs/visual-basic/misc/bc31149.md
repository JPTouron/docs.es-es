# <a name="duplicate-xml-attribute-attributename"></a>Atributo XML 'nombreDeAtributo' duplicado
Un atributo especificado para un literal de elemento XML coincide con el atributo definido previamente para el literal de elemento XML. Los nombres de atributo en los literales de elemento XML deben ser únicos.  
  
 **Identificador de error:** BC31149  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Cambie el nombre del atributo por un nombre que sea único para el literal de elemento XML.
