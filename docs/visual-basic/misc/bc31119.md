# <a name="end-removehandler-must-be-the-first-statement-on-a-line"></a>'End RemoveHandler' debe ser la primera instrucción de una línea

Una instrucción `End RemoveHandler` sigue a un separador de instrucciones de dos puntos (:). `End RemoveHandler` debe ser la única instrucción en su línea de código fuente.  
  
 **Identificador de error:** BC31119  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Separe las distintas instrucciones en líneas diferentes.
