# <a name="name-is-already-declared-as-a-parameter-of-this-method"></a>' \<name> ' ya se ha declarado como parámetro de este método
El nombre de parámetro especificado ya se ha declarado para este método.  
  
 **Identificador de error:** BC30734  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Cambie el nombre del parámetro para que sea único.
