# <a name="project-level-conditional-compilation-constant-is-not-valid-error"></a>Constante de compilación condicional de nivel de proyecto no válida: \<error>

Se especificó una opción de compilación condicional no válida para el proyecto.  
  
 **Id. de error:** BC31031  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Compruebe las constantes de compilación condicional utilizadas en este proyecto.
