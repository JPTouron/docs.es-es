# <a name="type-typename-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>No se puede usar el tipo '\<typename>' en un atributo porque no está declarado como 'Public'

Los atributos deben ser públicos.  
  
 **Identificador de error:** BC31516  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Agregue el modificador `Public` a la clase de atributo.
