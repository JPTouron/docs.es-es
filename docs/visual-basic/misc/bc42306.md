# <a name="xml-comment-tag-tag-is-not-permitted-on-a-element-language-element"></a>La etiqueta de comentario XML '\<tag>' no se permite en un elemento de idioma '\<element>'
\<tag>No se permite la etiqueta de comentario XML en un \<element> elemento ' '. Se omitirá el comentario XML.  
  
 Una etiqueta de comentario XML no es compatible con el elemento de lenguaje al que se aplica.  
  
 **Identificador de error:** BC42306  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite la etiqueta de comentario XML no compatible.
