# <a name="for-loop-control-variable-already-in-use-by-an-enclosing-for-loop"></a>Un bucle 'For' envolvente está usando ya la variable de control del bucle 'For'.
Todos los bucles `For` anidados deben usar variables de control únicas.  
  
 **Identificador de error:** BC30975  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Cambie uno de los nombres de variable de control.
