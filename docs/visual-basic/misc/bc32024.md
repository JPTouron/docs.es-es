# <a name="default-values-cannot-be-supplied-for-parameters-that-are-not-declared-optional"></a>Los valores predeterminados no se pueden proporcionar para parámetros no declarados como 'Optional'.
Los argumentos requeridos deben proporcionarse cuando se llama al procedimiento, y no toman valores predeterminados en la declaración.  
  
 **Identificador de error:** BC32024  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Declare el argumento para que sea `Optional` o elimine el valor predeterminado.
