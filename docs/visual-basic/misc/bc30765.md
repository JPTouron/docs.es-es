# <a name="all-projects-in-a-visual-basic-solution-must-target-the-same-platform-but-the-project-you-are-trying-to-add-targets-a-platform-other-than-the-one-specified-by-your-solution"></a>Todos los proyectos de una solución de Visual Basic deben tener como destino la misma plataforma, pero el proyecto que está intentando agregar tiene como destino una plataforma distinta de la especificada en la solución.

Todos los proyectos de una solución de Visual Basic deben tener como destino la misma plataforma, pero el proyecto que está agregando tiene como destino una plataforma distinta de la especificada en la solución.  
  
 **Identificador de error:** BC30765  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Agregue únicamente los proyectos que tienen como destino la misma plataforma especificada en la solución.
