# <a name="xml-comment-parameter-must-have-a-name-attribute"></a>El parámetro de comentario XML debe tener un atributo 'name'

El parámetro de comentario XML debe tener un atributo 'name'. Se omitirá el comentario XML.  
  
 A la etiqueta XML le falta un atributo `name` necesario.  
  
 **Identificador de error:** BC42308  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Proporcione el atributo `name` necesario para la etiqueta.
