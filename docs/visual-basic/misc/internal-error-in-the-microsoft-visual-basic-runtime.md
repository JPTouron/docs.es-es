# <a name="internal-error-in-the-microsoft-visual-basic-runtime"></a>Error interno en el tiempo de ejecución de Microsoft Visual Basic
Se ha producido un error interno en el tiempo de ejecución de Microsoft Visual Basic.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Anote las circunstancias en las que se ha produjo el error y llame a los Servicios de soporte técnico de Microsoft.
