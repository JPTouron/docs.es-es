# <a name="argument-cannot-be-an-empty-string"></a>El argumento no puede ser una cadena vacía
Se definió un argumento como una cadena vacía ("").  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Suministre un valor válido para el argumento.
