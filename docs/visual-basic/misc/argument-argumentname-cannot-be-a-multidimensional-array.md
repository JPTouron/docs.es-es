# <a name="argument-argumentname-cannot-be-a-multidimensional-array"></a>El argumento '\<argumentname>' no puede ser una matriz multidimensional

El rango de un argumento de matriz es mayor que 1, pero solo puede ser una matriz unidimensional.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Cambie el rango del argumento a 1.
