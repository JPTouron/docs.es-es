# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="0d8bf-102">Los operandos 'If' no pueden ser argumentos con nombre</span><span class="sxs-lookup"><span data-stu-id="0d8bf-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="0d8bf-103">No se pueden usar argumentos con nombre en los operandos del operador `If` .</span><span class="sxs-lookup"><span data-stu-id="0d8bf-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="0d8bf-104">El ejemplo siguiente causa este error:</span><span class="sxs-lookup"><span data-stu-id="0d8bf-104">The following example causes this error:</span></span>  
  
```vb

Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```

  
 <span data-ttu-id="0d8bf-106">**Identificador de error:** BC33105</span><span class="sxs-lookup"><span data-stu-id="0d8bf-106">**Error ID:** BC33105</span></span>
