# <a name="end-interface-must-be-preceded-by-a-matching-interface"></a>'End Interface' debe ir precedida de la instrucción 'Interface' correspondiente

Una instrucción `End Interface` se produce sin una instrucción `Interface` correspondiente. `End Interface` debe ir precedida de una instrucción `Interface` correspondiente.  
  
 **Identificador de error:** BC30252  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Compruebe si una instrucción `Interface` anterior está mal escrita o no es válida.  
  
2. Compruebe que los miembros `Interface` tengan el formato correcto.
