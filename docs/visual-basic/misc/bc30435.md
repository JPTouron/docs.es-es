# <a name="members-in-a-structure-cannot-be-declared-specifier"></a>Los miembros de una estructura no se pueden declarar como ' \<specifier> '

Usó un especificador que no es válido en un miembro dentro de una instrucción `Structure` .  
  
 **Id. de error:** BC30435  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el especificador.
