# <a name="typename-is-an-unsupported-type"></a>' \<typename> ' es un tipo no admitido
Visual Studio no puede representar el tipo especificado.  
  
 **Identificador de error:** BC30649  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Use un tipo compatible.  
  
- Desarrolle un tipo nuevo.
