# <a name="option-strict-on-does-not-allow-narrowing-in-implicit-type-conversions-between-method-methodname-and-delegate-delegatename"></a><span data-ttu-id="6df05-102">Option Strict on no permite restricciones en conversiones de tipo implícitas entre el método ' \<methodname> ' y el delegado ' \<delegatename> '</span><span class="sxs-lookup"><span data-stu-id="6df05-102">Option Strict On does not allow narrowing in implicit type conversions between method '\<methodname>' and delegate '\<delegatename>'</span></span>
<span data-ttu-id="6df05-103">Cuando se activa `Option Strict` , no se puede realizar una conversión de restricción entre el tipo de datos de un parámetro en un delegado y el parámetro correspondiente de una función o `Sub` asignado a una variable de ese tipo de delegado.</span><span class="sxs-lookup"><span data-stu-id="6df05-103">With `Option Strict` on, you cannot have a narrowing conversion between the data type of a parameter in a delegate and the corresponding parameter of a function or `Sub` assigned to a variable of that delegate type.</span></span> <span data-ttu-id="6df05-104">Por ejemplo, el delegado de función `Del` tiene un parámetro de tipo `Integer`, y las funciones `Conversion1`, `Conversion2`, y `Conversion3` tienen un parámetro de tipos numéricos diferentes.</span><span class="sxs-lookup"><span data-stu-id="6df05-104">For example, function delegate `Del` has one parameter of type `Integer`, and functions `Conversion1`, `Conversion2`, and `Conversion3` have one parameter of different numeric types.</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As String  
  
Function Conversion1(ByVal n As Integer) As String  
    Return "Valid"  
End Function  
  
Function Conversion2(ByVal n As Long) As String  
    Return "Valid"  
End Function  
  
Function Conversion3(ByVal n As Short) As String  
    Return "Not valid"  
End Function  
```  
  
 <span data-ttu-id="6df05-105">Como no hay una conversión de ampliación de `Integer` a `Integer` y a `Long`, las siguientes asignaciones son válidas.</span><span class="sxs-lookup"><span data-stu-id="6df05-105">Because there is a widening conversion from `Integer` to `Integer` and to `Long`, the following assignments are valid.</span></span>  
  
```vb  
' Valid.  
Dim funDel1 As Del = AddressOf Conversion1  
Dim funDel2 As Del = AddressOf Conversion2  
```  
  
 <span data-ttu-id="6df05-106">La conversión de `Integer` a `Short` es una conversión de reducción.</span><span class="sxs-lookup"><span data-stu-id="6df05-106">The conversion from `Integer` to `Short` is a narrowing conversion.</span></span> <span data-ttu-id="6df05-107">Por consiguiente, la siguiente asignación no es válida.</span><span class="sxs-lookup"><span data-stu-id="6df05-107">Therefore, the following assignment is not valid.</span></span>  
  
```vb  
' Not valid.  
Dim funDel3 As Del = AddressOf Conversion3  
```  
  
 <span data-ttu-id="6df05-108">Identificador de error: BC36663</span><span class="sxs-lookup"><span data-stu-id="6df05-108">Error ID: BC36663</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="6df05-109">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="6df05-109">To correct this error</span></span>
