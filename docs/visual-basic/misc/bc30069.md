# <a name="for-loop-control-variable-variablename-already-in-use-by-an-enclosing-for-loop"></a>Un bucle 'For' envolvente ya está usando la variable de control de bucle 'For' '\<variablename>'
Los bucles `For` anidados deben usar todos variables de control únicas.  
  
 **Identificador de error:** BC30069  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Cambie uno de los nombres de variable de control.
