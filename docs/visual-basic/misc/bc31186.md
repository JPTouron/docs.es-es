# <a name="-is-not-allowed"></a>no se permite ':'
no se permite ':'. No se pueden usar nombres XML completos en este contexto.  
  
 Se ha especificado un literal XML con un nombre completo en una ubicación no válida, por ejemplo, un destino en un literal de instrucción de procesamiento.  
  
 **Identificador de error:** BC31186  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el literal XML con el nombre completo o muévalo a una ubicación válida.  
  
     Para obtener información sobre qué caracteres son válidos en nombres XML, vea [Extensible Markup Language (XML) 1.0](https://www.w3.org/TR/xml).
