# <a name="local-variable-variablename-cannot-be-referred-to-before-it-is-declared"></a>No se puede hacer referencia a la variable local '\<variablename>' hasta que no esté declarada
Una variable dentro de un procedimiento se usa antes de su declaración.  
  
 **Identificador de error:** BC32000  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Mueva la declaración antes de la primera referencia a la variable.
