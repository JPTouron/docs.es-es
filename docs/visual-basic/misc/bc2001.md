# <a name="file-filename-could-not-be-found"></a>No se pudo encontrar el archivo \<filename>
No se encontró el archivo especificado. Puede que la causa sea haber facilitado una ruta de acceso de archivo incorrecta.  
  
 **Identificador de error:** BC2001  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Compruebe que el nombre de archivo y la ruta de acceso son correctos.
