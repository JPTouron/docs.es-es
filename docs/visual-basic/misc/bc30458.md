# <a name="types-are-not-available-in-this-context"></a>Los tipos no están disponibles en este contexto
Ha intentado hacer referencia a un tipo dentro de una directiva `#If` .  
  
 **Id. de error:** BC30458  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Quite la referencia.
