# <a name="error-structurename1-contains-structurename2"></a>\<error>: ' \<structurename1> ' contiene ' \<structurename2> '
Una definición de estructura circular se produce cuando una estructura se designa como uno de sus miembros.  
  
 **Id. de error:** BC30293  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Cambie el nombre del miembro de la estructura.
