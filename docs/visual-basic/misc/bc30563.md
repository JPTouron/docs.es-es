# <a name="name-is-ambiguous-in-the-application-objects-list"></a>' \<name> ' es ambiguo en los objetos de aplicación ' \<list> '
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El compilador Visual Basic no tiene ninguna regla de resolución de conflictos; debe eliminar la ambigüedad de los nombres. La lista proporcionada en el mensaje de error detalla los objetos de aplicación en los que el nombre es ambiguo.  
  
 **Identificador de error:** BC30563  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Use el nombre completo.
