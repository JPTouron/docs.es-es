# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>\<qualifiedmethod>no se puede obtener acceso a ' ' en este contexto porque es ' \<modifier> '
Ha intentado obtener acceso a un método que no es accesible en este contexto debido a un modificador que se le ha aplicado.  
  
 **Identificador de error:** BC30390  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Quite el modificador.
