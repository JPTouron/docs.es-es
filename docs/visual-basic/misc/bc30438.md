# <a name="constants-must-have-a-value"></a>Las constantes deben tener un valor
Declaró una constante sin asignarle un valor.  
  
 **Identificador de error:** BC30438  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Asigne un valor a la constante.
