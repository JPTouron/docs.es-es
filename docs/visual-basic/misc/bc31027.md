# <a name="unable-to-open-file-filename-error"></a>No se puede abrir el archivo ' \<filename> ': \<error>

Se produjo un error al abrir un archivo en este proyecto.  
  
 **Id. de error:** BC31027  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1. Compruebe que el archivo existe y que no está dañado o bloqueado por otra aplicación.
