# <a name="module-statement-must-end-with-a-matching-end-module"></a>La instrucción 'Module' debe terminar con la instrucción 'End Module' correspondiente
Los bloques de módulo deben terminar con una instrucción `End Module` .  
  
 **Identificador de error:** BC30625  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
- Agregue una instrucción `End Module` al final del bloque `Module` .
