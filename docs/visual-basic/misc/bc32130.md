# <a name="nullable-types-are-not-supported-because-the-current-version-of-mscorlibdll-does-not-define-the-type-nullableof-t"></a>No se admiten tipos que acepten valores NULL porque la versión actual de 'mscorlib.dll' no define el tipo 'Nullable(of T)'
La versión de Framework que está usando no admite tipos de valor que aceptan valores NULL.  
  
 **Identificador de error:** BC32130
