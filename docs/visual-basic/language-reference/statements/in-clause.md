# <a name="in-clause-visual-basic"></a>In (Cláusula, Visual Basic)

Especifica el grupo que la variable de bucle va a atravesar en un `For Each` bucle o especifica la colección que se va a consultar en una `From` `Join` cláusula, o `Group Join` .  
  
## <a name="remarks"></a>Comentarios  

 La `In` palabra clave se puede usar en los contextos siguientes:  
  
 [Instrucción For Each...Next](for-each-next-statement.md)  
  
 [Cláusula FROM](../queries/from-clause.md)  
  
 [Join (cláusula)](../queries/join-clause.md)  
  
 [Cláusula Group Join](../queries/group-join-clause.md)
