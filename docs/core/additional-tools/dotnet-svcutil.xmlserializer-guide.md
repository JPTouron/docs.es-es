## <a name="getting-started"></a><span data-ttu-id="7aaef-111">Introducción</span><span class="sxs-lookup"><span data-stu-id="7aaef-111">Getting started</span></span>

<span data-ttu-id="7aaef-112">Para usar `dotnet-svcutil.xmlserializer` en una aplicación de consola de .NET Core:</span><span class="sxs-lookup"><span data-stu-id="7aaef-112">To use `dotnet-svcutil.xmlserializer` in a .NET Core console application:</span></span>

1. <span data-ttu-id="7aaef-113">Crea un servicio WCF denominado "MyWCFService" mediante la plantilla predeterminada "Aplicación del servicio WCF" en .NET Framework.</span><span class="sxs-lookup"><span data-stu-id="7aaef-113">Create a WCF Service named 'MyWCFService' using the default template 'WCF Service Application' in .NET Framework.</span></span> <span data-ttu-id="7aaef-114">Agregue el atributo `[XmlSerializerFormat]` al método de servicio similar al siguiente:</span><span class="sxs-lookup"><span data-stu-id="7aaef-114">Add `[XmlSerializerFormat]` attribute on the service method like the following:</span></span>

   ```csharp

    {
        [XmlSerializerFormat]
        [OperationContract(Action = "http://tempuri.org/IService1/GetData", ReplyAction = "http://tempuri.org/IService1/GetDataResponse")]
        string GetData(int value);
    }
    ```

    ```dotnetcli
    dotnet new console --name MyWCFClient
    ```

    <span data-ttu-id="7aaef-117">Para asegurarse de que el proyecto está destinado a .NET Core 2.1 o una versión posterior, inspeccione el elemento XML `TargetFramework` del archivo de proyecto:</span><span class="sxs-lookup"><span data-stu-id="7aaef-117">To ensure your project is targeting .NET Core 2.1 or later, inspect the `TargetFramework` XML element in your project file:</span></span>

3. <span data-ttu-id="7aaef-118">Ejecute el comando siguiente para agregar una referencia de paquete a `System.ServiceModel.Http`:</span><span class="sxs-lookup"><span data-stu-id="7aaef-118">Add a package reference to `System.ServiceModel.Http` by running the following command:</span></span>

4. <span data-ttu-id="7aaef-119">Agregue el código del cliente WCF:</span><span class="sxs-lookup"><span data-stu-id="7aaef-119">Add the WCF Client code:</span></span>

    ```csharp

        class Program
        {
            static void Main(string[] args)
            {
                var myBinding = new BasicHttpBinding();
                var myEndpoint = new EndpointAddress("http://localhost:2561/Service1.svc"); //Fill your service url here
                var myChannelFactory = new ChannelFactory<IService1>(myBinding, myEndpoint);

        }

    [ServiceContract]
    public interface IService1
    {
        [XmlSerializerFormat]
        [OperationContract(Action = "http://tempuri.org/IService1/GetData", ReplyAction = "http://tempuri.org/IService1/GetDataResponse")]
        string GetData(int value);
    ```


    <span data-ttu-id="7aaef-121">Al ejecutar el comando se debería agregar una entrada al archivo de proyecto similar a la siguiente:</span><span class="sxs-lookup"><span data-stu-id="7aaef-121">Running the command should add an entry to your project file similar to this:</span></span>
