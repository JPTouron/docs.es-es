# <a name="endregion-c-reference"></a>#endregion (Referencia de C#)

`#endregion` marca el final de un bloque [#region](./preprocessor-region.md). Por ejemplo:  
  
```csharp
#region MyClass definition  
class MyClass
{  
    static void Main()
    {  
    }  
}  
#endregion  
```
