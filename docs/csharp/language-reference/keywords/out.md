# <a name="out-c-reference"></a>out (Referencia de C#)

Puede usar la palabra clave `out` en dos contextos:

- Como un [modificador de parámetro](out-parameter-modifier.md), que le permite pasar un argumento a un método mediante una referencia en lugar de mediante un valor.

- En [declaraciones de parámetro de tipo genérico](out-generic-modifier.md) para interfaces y delegados, que especifica que un parámetro de tipo es covariante.
