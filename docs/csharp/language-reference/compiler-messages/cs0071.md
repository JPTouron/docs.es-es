## <a name="example"></a><span data-ttu-id="7ca1d-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="7ca1d-108">Example</span></span>

 <span data-ttu-id="7ca1d-109">En el ejemplo siguiente se genera la advertencia CS0071.</span><span class="sxs-lookup"><span data-stu-id="7ca1d-109">The following sample generates CS0071.</span></span>

```csharp
// CS0071.cs
public delegate void MyEvent(object sender);

interface ITest
{
    event MyEvent Clicked;
}

class Test : ITest
{
    event MyEvent ITest.Clicked;  // CS0071

    // Try the following code instead.
    /*
    private MyEvent clicked;

    event MyEvent ITest.Clicked
    {
        add
        {
            clicked += value;
        }
        remove
        {
            clicked -= value;
        }
    }
    */
    public static void Main() { }
}
```
