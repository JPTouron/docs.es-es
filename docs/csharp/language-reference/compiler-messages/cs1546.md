## <a name="example"></a><span data-ttu-id="af136-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="af136-109">Example</span></span>

```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
