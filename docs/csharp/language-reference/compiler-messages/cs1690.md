# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="d3eba-103">Advertencia del compilador (nivel 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="d3eba-103">Compiler Warning (level 1) CS1690</span></span>

<span data-ttu-id="d3eba-104">El acceso a un miembro en 'member' podría provocar una excepción en tiempo de ejecución, ya que es un campo de una clase de serialización por referencia</span><span class="sxs-lookup"><span data-stu-id="d3eba-104">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="d3eba-105">Esta advertencia se produce cuando intenta llamar a un método, propiedad o indexador en un miembro de una clase que deriva de <xref:System.MarshalByRefObject>, y el miembro es un tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="d3eba-105">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="d3eba-106">Los objetos que se heredan de `MarshalByRefObject` están diseñados normalmente para calcularse mediante referencia en un dominio de aplicación.</span><span class="sxs-lookup"><span data-stu-id="d3eba-106">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="d3eba-107">Si alguna vez algún código intenta tener acceso al miembro de tipo de valor directamente de dicho objeto en un dominio de aplicación, se producirá una <xref:System.InvalidOperationException> en tiempo de ejecución.</span><span class="sxs-lookup"><span data-stu-id="d3eba-107">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime <xref:System.InvalidOperationException> will occur.</span></span> <span data-ttu-id="d3eba-108">Para resolver la advertencia, primero copie el miembro en una variable local y llame al método en esa variable.</span><span class="sxs-lookup"><span data-stu-id="d3eba-108">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>
  
 <span data-ttu-id="d3eba-109">En el ejemplo siguiente se genera el error CS1690:</span><span class="sxs-lookup"><span data-stu-id="d3eba-109">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;

class WarningCS1690 : MarshalByRefObject
{
    int i = 5;

    public static void Main()
    {
        AppDomain domain = AppDomain.CreateDomain("MyDomain");
        Type t = typeof(WarningCS1690);
        WarningCS1690 e = (WarningCS1690)domain.CreateInstanceAndUnwrap(t.Assembly.FullName,t.FullName);

        e.i.ToString(); // CS1690

        // OK  
        int i = e.i;
        i.ToString();
        e.i = i;
    }
}
```
