# <a name="compiler-error-cs8400"></a>Error del compilador CS8400

La característica "feature" no está disponible en C# 8.0. Use una versión de lenguaje \<version> o superior.

## <a name="to-correct-this-error"></a>Para corregir este error

Actualice el proyecto al menos a la versión mencionada en el error.
