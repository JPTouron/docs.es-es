# <a name="compiler-error-cs0233"></a><span data-ttu-id="c9589-103">Error del compilador CS0233</span><span class="sxs-lookup"><span data-stu-id="c9589-103">Compiler Error CS0233</span></span>
<span data-ttu-id="c9589-104">"identifier" no tiene un tamaño predefinido, por lo que sizeof solo se puede usar en un contexto de unsafe.</span><span class="sxs-lookup"><span data-stu-id="c9589-104">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context</span></span>
  
 <span data-ttu-id="c9589-105">Sin el contexto de [unsafe](../keywords/unsafe.md), el operador [sizeof](../operators/sizeof.md) solo se puede usar para los tipos cuyo tamaño es una constante de tiempo de compilación.</span><span class="sxs-lookup"><span data-stu-id="c9589-105">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="c9589-106">Si recibe este error, use un contexto de unsafe.</span><span class="sxs-lookup"><span data-stu-id="c9589-106">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="c9589-107">En el ejemplo siguiente se genera el error CS0233:</span><span class="sxs-lookup"><span data-stu-id="c9589-107">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
