## <a name="example"></a><span data-ttu-id="4220c-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="4220c-109">Example</span></span>

<span data-ttu-id="4220c-110">En el ejemplo siguiente, CS0019 se genera en tres lugares porque [bool](../builtin-types/bool.md) en C# no puede convertirse en [int](../builtin-types/integral-numeric-types.md). CS0019 también se genera cuando el operador de resta `-` se aplica a una cadena.</span><span class="sxs-lookup"><span data-stu-id="4220c-110">In the following example, CS0019 is generated in three places because [bool](../builtin-types/bool.md) in C# is not convertible to [int](../builtin-types/integral-numeric-types.md). CS0019 is also generated when the subtraction operator `-` is applied to a string.</span></span> <span data-ttu-id="4220c-111">El operador de suma `+` puede usarse con operandos de cadena porque ese operador está sobrecargado mediante la clase `String` para realizar la concatenación de cadenas.</span><span class="sxs-lookup"><span data-stu-id="4220c-111">The addition operator `+` can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>

```csharp
static void Main()
{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }

    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```
