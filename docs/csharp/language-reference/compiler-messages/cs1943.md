# <a name="compiler-error-cs1943"></a><span data-ttu-id="00ca1-103">Error del compilador CS1943</span><span class="sxs-lookup"><span data-stu-id="00ca1-103">Compiler Error CS1943</span></span>

<span data-ttu-id="00ca1-104">No se permiten expresiones de tipo 'type' en una cláusula from siguiente incluida en una expresión de consulta con el tipo de origen 'type'.</span><span class="sxs-lookup"><span data-stu-id="00ca1-104">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="00ca1-105">No se pudo realizar la inferencia de tipos en la llamada a 'method'.</span><span class="sxs-lookup"><span data-stu-id="00ca1-105">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="00ca1-106">Todas las variables de rango deben representar tipos que se pueden consultar.</span><span class="sxs-lookup"><span data-stu-id="00ca1-106">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="00ca1-107">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="00ca1-107">To correct this error</span></span>  
  
1. <span data-ttu-id="00ca1-108">Asegúrese de que el tipo es un tipo consultable que implementa `IEnumerable`, `IEnumerable<T>` o una interfaz derivada, o cualquier otro tipo que tenga un modelo de consulta definido para este.</span><span class="sxs-lookup"><span data-stu-id="00ca1-108">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2. <span data-ttu-id="00ca1-109">Si el tipo es un `IEnumerable` no genérico, proporcione un tipo explícito en la variable de rango.</span><span class="sxs-lookup"><span data-stu-id="00ca1-109">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="00ca1-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="00ca1-110">Example</span></span>  

 <span data-ttu-id="00ca1-111">El código siguiente genera el error CS1943:</span><span class="sxs-lookup"><span data-stu-id="00ca1-111">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
