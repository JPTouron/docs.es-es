# <a name="compiler-error-cs0310"></a><span data-ttu-id="71329-103">Error del compilador CS0310</span><span class="sxs-lookup"><span data-stu-id="71329-103">Compiler Error CS0310</span></span>
<span data-ttu-id="71329-104">El tipo 'typename' debe ser un tipo no abstracto con un constructor público sin parámetros para poder usarlo como parámetro 'parameter' en el tipo o método genérico 'generic'</span><span class="sxs-lookup"><span data-stu-id="71329-104">The type 'typename' must be a non-abstract type with a public parameterless constructor in order to use it as parameter 'parameter' in the generic type or method 'generic'</span></span>  
  
 <span data-ttu-id="71329-105">El tipo o método genérico define una nueva restricción en su cláusula where, por lo que cualquier tipo debe tener un constructor sin parámetros público para usarse como un argumento de tipo para ese método o tipo genérico.</span><span class="sxs-lookup"><span data-stu-id="71329-105">The generic type or method defines a new constraint in its where clause, so any type must have a public parameterless constructor in order to be used as a type argument for that generic type or method.</span></span> <span data-ttu-id="71329-106">Para evitar este error, asegúrese de que el tipo tenga el constructor correcto o modifique la cláusula de restricción del método o tipo genérico.</span><span class="sxs-lookup"><span data-stu-id="71329-106">To avoid this error, make sure that the type has the correct constructor, or modify the constraint clause of the generic type or method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="71329-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="71329-107">Example</span></span>  
 <span data-ttu-id="71329-108">En el ejemplo siguiente se genera la advertencia CS0310:</span><span class="sxs-lookup"><span data-stu-id="71329-108">The following sample generates CS0310:</span></span>  
  
```csharp  
// CS0310.cs  
using System;  
  
class G<T> where T : new()  
{  
    T t;  
  
    public G()  
    {  
        t = new T();  
        Console.WriteLine(t);  
    }  
}  
  
class B  
{  
    private B() { }  
    // Try this instead:  
    // public B() { }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
        G<B> g = new G<B>();   // CS0310  
        Console.WriteLine(g.ToString());  
    }  
}  
```
