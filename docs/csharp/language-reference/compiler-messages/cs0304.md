# <a name="compiler-error-cs0304"></a><span data-ttu-id="aeba1-103">Error del compilador CS0304</span><span class="sxs-lookup"><span data-stu-id="aeba1-103">Compiler Error CS0304</span></span>

<span data-ttu-id="aeba1-104">No se puede crear ninguna instancia del tipo de variable "type" porque no tiene la restricción new()</span><span class="sxs-lookup"><span data-stu-id="aeba1-104">Cannot create an instance of the variable type 'type' because it does not have the new() constraint</span></span>  
  
 <span data-ttu-id="aeba1-105">Si al implementar una clase genérica quiere usar la palabra clave `new` para crear una nueva instancia de cualquier tipo que se proporcione para un parámetro de tipo `T`, es preciso aplicar la [restricción new()](../keywords/new-constraint.md) a `T` en la declaración de clase, tal y como se muestra en el siguiente ejemplo.</span><span class="sxs-lookup"><span data-stu-id="aeba1-105">When you implement a generic class, and you want to use the `new` keyword to create a new instance of any type that is supplied for a type parameter `T`, you must apply the [new() constraint](../keywords/new-constraint.md) to `T` in the class declaration, as shown in the following example.</span></span>  
  
```csharp  
class C<T> where T : new()  
```  
  
 <span data-ttu-id="aeba1-106">La restricción `new()` aplica la seguridad de tipos garantizando que cualquier tipo concreto que se proporcione para `T` tiene un constructor sin parámetros predeterminado.</span><span class="sxs-lookup"><span data-stu-id="aeba1-106">The `new()` constraint enforces type safety by guaranteeing that any concrete type that is supplied for `T` has a default, parameterless constructor.</span></span> <span data-ttu-id="aeba1-107">El error CS0304 se produce si se intenta usar el operador `new` en el cuerpo de la clase para crear una instancia del parámetro de tipo `T` cuando `T` no especifica la restricción `new()`.</span><span class="sxs-lookup"><span data-stu-id="aeba1-107">CS0304 occurs if you attempt to use the `new` operator in the body of the class to create an instance of type parameter `T` when `T` does not specify the `new()` constraint.</span></span> <span data-ttu-id="aeba1-108">En el lado cliente, si el código intenta crear instancias de la clase genérica con un tipo que no tiene ningún constructor sin parámetros, el código generará el [Error del compilador CS0310](./cs0310.md).</span><span class="sxs-lookup"><span data-stu-id="aeba1-108">On the client side, if code attempts to instantiate the generic class with a type that has no parameterless constructor, that code will generate [Compiler Error CS0310](./cs0310.md).</span></span>  
  
 <span data-ttu-id="aeba1-109">En el ejemplo siguiente se genera el error CS0304.</span><span class="sxs-lookup"><span data-stu-id="aeba1-109">The following example generates CS0304.</span></span>  
  
```csharp  
// CS0304.cs  
// Compile with: /target:library.  
class C<T>  
{  
    // The following line generates CS0304.  
    T t = new T();  
}  
```  
  
 <span data-ttu-id="aeba1-110">Tampoco se permite el operador `new` en los métodos de clase.</span><span class="sxs-lookup"><span data-stu-id="aeba1-110">The `new` operator also is not allowed in methods of the class.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T>  
{  
    public void ExampleMethod()  
    {  
        // The following line generates CS0304.  
        T t = new T();  
    }  
}  
```  
  
 <span data-ttu-id="aeba1-111">Para evitar el error, declare la clase usando la restricción `new()`, tal y como se muestra en el ejemplo siguiente.</span><span class="sxs-lookup"><span data-stu-id="aeba1-111">To avoid the error, declare the class by using the `new()` constraint, as shown in the following example.</span></span>  
  
```csharp  
// Compile with: /target:library.  
class C<T> where T : new()  
{  
    T t = new T();  
  
    public void ExampleMethod()  
    {  
        T t = new T();  
    }  
}  
```
