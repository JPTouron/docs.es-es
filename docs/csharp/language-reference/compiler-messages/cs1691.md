# <a name="compiler-warning-level-1-cs1691"></a>Advertencia del compilador (nivel 1) CS1691
'number' no es un número de advertencia válido  
  
 Un número que se ha pasado a la directiva del preprocesador [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) no era un número de advertencia válido. Compruebe que el número representa una advertencia, no un error u otra secuencia de caracteres.  
  
## <a name="example"></a>Ejemplo  
 En el ejemplo siguiente se genera el error CS1691.  
  
```csharp  
// CS1691.cs  
public class C  
{  
    int i = 1;  
    public static void Main()  
    {  
        C myC = new C();  
#pragma warning disable 151  // CS1691  
// Try the following line instead:  
// #pragma warning disable 1645
        myC.i++;  
#pragma warning restore 151  // CS1691  
// Try the following line instead:  
// #pragma warning restore 1645
    }  
}  
```
