## <a name="example"></a><span data-ttu-id="b67b4-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="b67b4-109">Example</span></span>

 <span data-ttu-id="b67b4-110">En el ejemplo siguiente se genera la advertencia CS0592:</span><span class="sxs-lookup"><span data-stu-id="b67b4-110">The following sample generates CS0592:</span></span>  

```csharp

[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.

{  
    public static void Main()  
    {  
    }  
}  
```
