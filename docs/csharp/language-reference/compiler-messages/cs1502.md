<span data-ttu-id="ac466-104">La mejor coincidencia de método sobrecargado para 'name' tiene algunos argumentos no válidos</span><span class="sxs-lookup"><span data-stu-id="ac466-104">The best overloaded method match for 'name' has some invalid arguments</span></span>

 <span data-ttu-id="ac466-107">Para solucionar este problema, realice una de las siguientes acciones:</span><span class="sxs-lookup"><span data-stu-id="ac466-107">To resolve this problem, do one of the following:</span></span>
