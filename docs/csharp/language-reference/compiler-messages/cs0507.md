# <a name="compiler-error-cs0507"></a><span data-ttu-id="ec62c-103">Error del compilador CS0507</span><span class="sxs-lookup"><span data-stu-id="ec62c-103">Compiler Error CS0507</span></span>

<span data-ttu-id="ec62c-104">'function1' : no se pueden cambiar los modificadores de acceso al invalidar el miembro heredado 'access' de acceso 'function2'</span><span class="sxs-lookup"><span data-stu-id="ec62c-104">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="ec62c-105">Se realizó un intento de cambiar la especificación de acceso en un reemplazo de método.</span><span class="sxs-lookup"><span data-stu-id="ec62c-105">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ec62c-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="ec62c-106">Example</span></span>  

 <span data-ttu-id="ec62c-107">En el ejemplo siguiente se genera la advertencia CS0507.</span><span class="sxs-lookup"><span data-stu-id="ec62c-107">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ec62c-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="ec62c-108">Example</span></span>  

 <span data-ttu-id="ec62c-109">El error CS0507 también puede aparecer si una clase intenta reemplazar un método marcado como `protected internal` definido en metadatos a los que se hace referencia.</span><span class="sxs-lookup"><span data-stu-id="ec62c-109">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="ec62c-110">En esta situación, el método que reemplaza se debería marcar como `protected`.</span><span class="sxs-lookup"><span data-stu-id="ec62c-110">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs

abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ec62c-111">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="ec62c-111">Example</span></span>  

 <span data-ttu-id="ec62c-112">En el ejemplo siguiente se genera la advertencia CS0507.</span><span class="sxs-lookup"><span data-stu-id="ec62c-112">The following sample generates CS0507.</span></span>
