# <a name="compiler-error-cs1708"></a><span data-ttu-id="c5613-103">Error del compilador CS1708</span><span class="sxs-lookup"><span data-stu-id="c5613-103">Compiler Error CS1708</span></span>
<span data-ttu-id="c5613-104">Solo se puede tener acceso a los búferes de tamaño fijo mediante variables locales o campos</span><span class="sxs-lookup"><span data-stu-id="c5613-104">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="c5613-105">Una de las nuevas características en C# 2.0 es la capacidad de definir una matriz en línea dentro de un `struct`.</span><span class="sxs-lookup"><span data-stu-id="c5613-105">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="c5613-106">A este tipo de matrices solo se puede tener acceso mediante variables o campos locales, y no se puede hacer referencia a ellas como valores intermedios en el lado izquierdo de una expresión.</span><span class="sxs-lookup"><span data-stu-id="c5613-106">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="c5613-107">Además, solo pueden tener acceso a las matrices los campos que son `static` o `readonly`.</span><span class="sxs-lookup"><span data-stu-id="c5613-107">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="c5613-108">Para resolver este error, defina una variable de matriz y asigne la matriz en línea a la variable.</span><span class="sxs-lookup"><span data-stu-id="c5613-108">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="c5613-109">O quite el modificador `static` o `readonly` del campo que representa la matriz en línea.</span><span class="sxs-lookup"><span data-stu-id="c5613-109">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c5613-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="c5613-110">Example</span></span>  
 <span data-ttu-id="c5613-111">En el ejemplo siguiente se genera la advertencia CS1708.</span><span class="sxs-lookup"><span data-stu-id="c5613-111">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
