# <a name="compiler-error-cs0001"></a>Error del compilador CS0001
Error interno del compilador

 Intente determinar si el compilador está fallando porque no puede analizar una sintaxis inesperada. Si recibe este error varias veces, póngase en contacto con Microsoft.
