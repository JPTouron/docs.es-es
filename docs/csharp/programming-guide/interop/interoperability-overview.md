## <a name="c-interop"></a><span data-ttu-id="56535-113">Interoperabilidad de C++</span><span class="sxs-lookup"><span data-stu-id="56535-113">C++ Interop</span></span>  
 <span data-ttu-id="56535-114">Puede usar la interoperabilidad de C++, también conocida como It Just Works (IJW), para encapsular una clase de C++ nativa de modo que el código creado en C# o en otro lenguaje de .NET pueda consumirla.</span><span class="sxs-lookup"><span data-stu-id="56535-114">You can use C++ interop, also known as It Just Works (IJW), to wrap a native C++ class so that it can be consumed by code that is authored in C# or another .NET language.</span></span> <span data-ttu-id="56535-115">Para ello, escriba código de C++ para encapsular un componente DLL o COM nativo.</span><span class="sxs-lookup"><span data-stu-id="56535-115">To do this, you write C++ code to wrap a native DLL or COM component.</span></span> <span data-ttu-id="56535-116">A diferencia de otros lenguajes de .NET, Visual C++ cuenta con compatibilidad de interoperabilidad que permite que haya código administrado y no administrado en la misma aplicación, e incluso en el mismo archivo.</span><span class="sxs-lookup"><span data-stu-id="56535-116">Unlike other .NET languages, Visual C++ has interoperability support that enables managed and unmanaged code to be located in the same application and even in the same file.</span></span> <span data-ttu-id="56535-117">Después, compile el código de C++ mediante el modificador del compilador **/clr** para generar un ensamblado administrado.</span><span class="sxs-lookup"><span data-stu-id="56535-117">You then build the C++ code by using the **/clr** compiler switch to produce a managed assembly.</span></span> <span data-ttu-id="56535-118">Finalmente, agregue una referencia al ensamblado en el proyecto de C# y use los objetos encapsulados igual que usaría otras clases administradas.</span><span class="sxs-lookup"><span data-stu-id="56535-118">Finally, you add a reference to the assembly in your C# project and use the wrapped objects just as you would use other managed classes.</span></span>

 <span data-ttu-id="56535-120">Puede usar un componente COM de un proyecto de C#.</span><span class="sxs-lookup"><span data-stu-id="56535-120">You can consume a COM component from a C# project.</span></span> <span data-ttu-id="56535-121">Los pasos generales son los siguientes:</span><span class="sxs-lookup"><span data-stu-id="56535-121">The general steps are as follows:</span></span>

 <span data-ttu-id="56535-132">Para obtener más información, vea [Exponer componentes COM en .NET Framework](../../../framework/interop/exposing-com-components.md).</span><span class="sxs-lookup"><span data-stu-id="56535-132">For more information, see [Exposing COM Components to the .NET Framework](../../../framework/interop/exposing-com-components.md).</span></span>
