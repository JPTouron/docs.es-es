## <a name="expression-body-definitions"></a><span data-ttu-id="37391-186">Definiciones de cuerpos de expresión</span><span class="sxs-lookup"><span data-stu-id="37391-186">Expression body definitions</span></span>

<span data-ttu-id="37391-187">Es habitual tener definiciones de método que simplemente hacen las devoluciones de forma inmediata con el resultado de una expresión, o que tienen una sola instrucción como cuerpo del método.</span><span class="sxs-lookup"><span data-stu-id="37391-187">It is common to have method definitions that simply return immediately with the result of an expression, or that have a single statement as the body of the method.</span></span> <span data-ttu-id="37391-188">Hay un acceso directo de sintaxis para definir este método mediante `=>`:</span><span class="sxs-lookup"><span data-stu-id="37391-188">There is a syntax shortcut for defining such methods using `=>`:</span></span>
