## <a name="example"></a><span data-ttu-id="e5f23-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="e5f23-108">Example</span></span>  
 <span data-ttu-id="e5f23-109">El ejemplo siguiente genera el error CS0838:</span><span class="sxs-lookup"><span data-stu-id="e5f23-109">The following example generates CS0838:</span></span>  
  
```csharp  
// cs0838.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Main()  
        {  
  
            Expression<Func<int[,]>> expr =  
                () => new int[2, 2] { { 1, 2 }, { 3, 4 } }; // CS0838  
  
            // try the following 2 lines instead  
            int[,] nums = new int[2, 2] { { 1, 2 }, { 3, 4 } };  
            Expression<Func<int[,]>> expr2 = () => nums;
  
            return 1;  
        }  
    }  
}  
```
