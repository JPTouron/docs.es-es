# <a name="compiler-error-cs0506"></a><span data-ttu-id="4b694-103">Error del compilador CS0506</span><span class="sxs-lookup"><span data-stu-id="4b694-103">Compiler Error CS0506</span></span>

<span data-ttu-id="4b694-104">'function1': no se puede invalidar el miembro heredado 'function2' porque no está marcado como "virtual", "abstract" ni "override"</span><span class="sxs-lookup"><span data-stu-id="4b694-104">'function1' : cannot override inherited member 'function2' because it is not marked "virtual", "abstract", or "override"</span></span>  
  
 <span data-ttu-id="4b694-105">Se reemplazó un método que no estaba marcado explícitamente como **virtual**, **abstract**ni `override`.</span><span class="sxs-lookup"><span data-stu-id="4b694-105">A method was overridden that was not explicitly marked as **virtual**, **abstract**, or `override`.</span></span>  
  
 <span data-ttu-id="4b694-106">El ejemplo siguiente genera la advertencia CS0506:</span><span class="sxs-lookup"><span data-stu-id="4b694-106">The following sample generates CS0506:</span></span>  
  
```csharp  
// CS0506.cs  
namespace MyNameSpace  
{  
   abstract public class ClassX  
   {  
      public int i = 0;  
  
      public int f()  
      {  
         return 0;  
      }  
      // Try the following definition for f() instead:  
      // abstract public int f();  
   }  
  
   public class ClassY : ClassX  
   {  
      public override int f()   // CS0506  
      {  
         return 0;  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
