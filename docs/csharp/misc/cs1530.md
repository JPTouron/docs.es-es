# <a name="compiler-error-cs1530"></a>Error del compilador CS1530

No se permite el uso de la palabra clave 'new' en elementos definidos en un espacio de nombres  
  
 No es necesario especificar la palabra clave [new](../language-reference/keywords/new-modifier.md) en ninguna construcci√≥n que se encuentre en un [espacio de nombres](../language-reference/keywords/namespace.md).  
  
 El ejemplo siguiente genera la advertencia CS1530:  
  
```csharp  
// CS1530.cs  
namespace a  
{  
   new class i   // CS1530  
   {  
   }  
  
   // try the following instead  
   class ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
