# <a name="compiler-error-cs0466"></a><span data-ttu-id="abc06-103">Error del compilador CS0466</span><span class="sxs-lookup"><span data-stu-id="abc06-103">Compiler Error CS0466</span></span>
<span data-ttu-id="abc06-104">'method1' no debe tener un parámetro params porque 'method2' tampoco lo tiene</span><span class="sxs-lookup"><span data-stu-id="abc06-104">'method1' should not have a params parameter since 'method2' does not</span></span>  
  
 <span data-ttu-id="abc06-105">No se puede usar el parámetro `params` en un miembro de clase si la interfaz implementada no lo usa.</span><span class="sxs-lookup"><span data-stu-id="abc06-105">You cannot use `params` parameter on a class member if the implemented interface doesn't use it.</span></span>  
  
## <a name="example"></a><span data-ttu-id="abc06-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="abc06-106">Example</span></span>  
 <span data-ttu-id="abc06-107">El ejemplo siguiente genera la advertencia CS0466:</span><span class="sxs-lookup"><span data-stu-id="abc06-107">The following sample generates CS0466.</span></span>  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
