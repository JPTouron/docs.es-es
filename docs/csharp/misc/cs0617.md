# <a name="compiler-error-cs0617"></a><span data-ttu-id="dddcc-103">Error del compilador CS0617</span><span class="sxs-lookup"><span data-stu-id="dddcc-103">Compiler Error CS0617</span></span>

<span data-ttu-id="dddcc-104">'reference' no es un argumento de atributo con nombre válido porque no es un tipo de parámetro de atributo válido</span><span class="sxs-lookup"><span data-stu-id="dddcc-104">'reference' is not a valid named attribute argument because it is not a valid attribute parameter type</span></span>  
  
 <span data-ttu-id="dddcc-105">Se ha intentado obtener acceso a un miembro [private](../language-reference/keywords/private.md) de una clase de atributo.</span><span class="sxs-lookup"><span data-stu-id="dddcc-105">An attempt was made to access a [private](../language-reference/keywords/private.md) member of an attribute class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dddcc-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="dddcc-106">Example</span></span>  

 <span data-ttu-id="dddcc-107">El ejemplo siguiente genera la advertencia CS0617.</span><span class="sxs-lookup"><span data-stu-id="dddcc-107">The following sample generates CS0617.</span></span>  
  
```csharp  
// CS0617.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Struct |
                AttributeTargets.Class |  
                AttributeTargets.Interface)]  
public class MyClass : Attribute  
{  
   public int Name;  
  
   public MyClass (int sName)  
   {  
      Name = sName;  
      Bad = -1;  
      Bad2 = -1;  
   }  
  
   public readonly int Bad;  
   public int Bad2;  
}  
  
[MyClass(5, Bad=0)] class Class1 {}   // CS0617  
[MyClass(5, Bad2=0)] class Class2 {}  
```
