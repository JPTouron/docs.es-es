# <a name="compiler-error-cs0517"></a>Error del compilador CS0517

'class' no tiene clase base y no puede llamar a un constructor base  
  
 CS0517 solo se puede producir cuando el tiempo de ejecución de .NET compila el código fuente de la clase de objeto, que es la única clase que no tiene clase base.
