# <a name="compiler-error-cs1588"></a>Error del compilador CS1588
No se puede determinar el directorio de Common Language Runtime: 'motivo'  
  
 El compilador no puede recuperar el directorio en el que está instalado el entorno de ejecución de .NET. Este error indica que Common Language Runtime está instalado correctamente.
