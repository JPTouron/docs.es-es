 <span data-ttu-id="5ed2f-105">Error al crear a un delegado porque no se usó la signatura correcta.</span><span class="sxs-lookup"><span data-stu-id="5ed2f-105">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="5ed2f-106">Las instancias de un delegado deben declararse con la misma signatura que la declaración de delegado.</span><span class="sxs-lookup"><span data-stu-id="5ed2f-106">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="5ed2f-107">Puede resolver este error ajustando la signatura del método o del delegado.</span><span class="sxs-lookup"><span data-stu-id="5ed2f-107">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="5ed2f-108">Para obtener más información, vea [Delegados (Guía de programación de C#)](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="5ed2f-108">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="5ed2f-109">El ejemplo siguiente genera la advertencia CS0123.</span><span class="sxs-lookup"><span data-stu-id="5ed2f-109">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
