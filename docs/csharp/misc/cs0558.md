# <a name="compiler-error-cs0558"></a>Error del compilador CS0558
El operador 'operator' definido por el usuario debe declararse como static y public  
  
 Los **modificadores** de acceso **static** y [public](../language-reference/keywords/index.md) deben especificarse ambos en los operadores definidos por el usuario.  
  
 El ejemplo siguiente genera la advertencia CS0558:  
  
```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
