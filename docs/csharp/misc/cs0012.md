 <span data-ttu-id="0df6c-109">La siguiente secuencia de compilaciones dará como resultado CS0012:</span><span class="sxs-lookup"><span data-stu-id="0df6c-109">The following sequence of compilations will result in CS0012:</span></span>

  
 <span data-ttu-id="0df6c-110">A continuación:</span><span class="sxs-lookup"><span data-stu-id="0df6c-110">Then:</span></span>  
  
```csharp

{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="0df6c-111">A continuación:</span><span class="sxs-lookup"><span data-stu-id="0df6c-111">Then:</span></span>

```csharp  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```
