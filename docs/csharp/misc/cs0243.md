<span data-ttu-id="e9bf0-104">El atributo Conditional no es válido en 'método' porque es un método de invalidación.</span><span class="sxs-lookup"><span data-stu-id="e9bf0-104">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="e9bf0-105"><xref:System.Diagnostics.ConditionalAttribute>No se permite el atributo en un método marcado con la palabra clave [override](../language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="e9bf0-105">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="e9bf0-106">Para obtener más información, consulte [saber cuándo usar las palabras clave override y New](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="e9bf0-106">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="e9bf0-107">El compilador nunca se enlaza a métodos de invalidación.</span><span class="sxs-lookup"><span data-stu-id="e9bf0-107">The compiler never binds to override methods.</span></span> <span data-ttu-id="e9bf0-108">Solo se enlaza al método base y el Common Language Runtime llama a la invalidación, según corresponda.</span><span class="sxs-lookup"><span data-stu-id="e9bf0-108">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="e9bf0-109">El código siguiente genera CS0243:</span><span class="sxs-lookup"><span data-stu-id="e9bf0-109">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
