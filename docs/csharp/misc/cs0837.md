 <span data-ttu-id="2bbee-105">Las expresiones lambda, los métodos anónimos y los grupos de métodos no se pueden usar en el lado izquierdo de [is](../language-reference/operators/type-testing-and-cast.md#is-operator) o [as](../language-reference/operators/type-testing-and-cast.md#as-operator).</span><span class="sxs-lookup"><span data-stu-id="2bbee-105">Lambda expressions, anonymous methods and method groups may not be used on the left side of [is](../language-reference/operators/type-testing-and-cast.md#is-operator) or [as](../language-reference/operators/type-testing-and-cast.md#as-operator).</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="2bbee-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="2bbee-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="2bbee-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="2bbee-110">Example</span></span>

<span data-ttu-id="2bbee-111">El ejemplo siguiente genera el error CS0837:</span><span class="sxs-lookup"><span data-stu-id="2bbee-111">The following example generates CS0837:</span></span>

```csharp
// cs0837.cs
namespace TestNamespace
{
    public delegate void Del();

    class Test
    {
        static int Main()
        {
            bool b1 = (() => { }) is Del;   // CS0837
            bool b2 = delegate() { } is Del;// CS0837
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837
            return 1;
        }
    }
}
```
