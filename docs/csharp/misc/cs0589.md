# <a name="compiler-error-cs0589"></a>Error del compilador CS0589

Error interno del compilador: fase 'PARSE'

 Intente determinar si el compilador está fallando porque no puede analizar una sintaxis inesperada. Si no es así, [hable con nosotros](/visualstudio/ide/feedback-options).

> [!NOTE]
> Este error del compilador ya no se usa en Roslyn.
