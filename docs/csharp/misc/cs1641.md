<span data-ttu-id="eb42f-104">Un campo de búfer de tamaño fijo debe tener el especificador de tamaño de matriz detrás del nombre de campo</span><span class="sxs-lookup"><span data-stu-id="eb42f-104">A fixed size buffer field must have the array size specifier after the field name</span></span>

 <span data-ttu-id="eb42f-107">El ejemplo siguiente genera la advertencia CS1641:</span><span class="sxs-lookup"><span data-stu-id="eb42f-107">The following sample generates CS1641:</span></span>  
  
```csharp  
// CS1641.cs  
// compile with: /unsafe /target:library  
unsafe struct S {  
   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
