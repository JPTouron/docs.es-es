# <a name="compiler-error-cs0196"></a>Error del compilador CS0196
Un puntero solo puede estar indizado por un valor  
  
 Un puntero no puede tener varios índices. Para obtener más información, vea [tipos de puntero](../programming-guide/unsafe-code-pointers/pointer-types.md)  
  
 El ejemplo siguiente genera la advertencia CS0196:  
  
```csharp  
// CS0196.cs  
public class MyClass  
{  
   public static void Main ()  
   {  
      int *i = null;  
      int j = 0;  
      j = i[1,2];   // CS0196  
      // try the following line instead  
      // j = i[1];  
   }  
}  
```
