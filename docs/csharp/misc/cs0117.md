## <a name="example"></a>Ejemplo  

 El ejemplo siguiente genera la advertencia CS0117:  
  
```csharp  
// CS0117.cs  
public class BaseClass { }  
  
    public class base021 : BaseClass  
    {  
        public void TestInt()  
        {  
            int i = base.someMember; //CS0117  
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
