# <a name="compiler-error-cs1599"></a><span data-ttu-id="89970-103">Error del compilador CS1599</span><span class="sxs-lookup"><span data-stu-id="89970-103">Compiler Error CS1599</span></span>

 <span data-ttu-id="89970-105">Algunos tipos de la biblioteca de clases de .NET Framework, como <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> y <xref:System.ArgIterator> , no pueden usarse como tipos de valor devuelto, ya que estos podrían emplearse para realizar operaciones no seguras.</span><span class="sxs-lookup"><span data-stu-id="89970-105">Some types in the .NET Framework class library, for example, <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> and <xref:System.ArgIterator> cannot be used as return types because they can potentially be used to perform unsafe operations.</span></span>  
  
 <span data-ttu-id="89970-106">El ejemplo siguiente genera la advertencia CS1599:</span><span class="sxs-lookup"><span data-stu-id="89970-106">The following sample generates CS1599:</span></span>  
  
```csharp

{  
   public static void Main()  
   {  
   }  
  
   public TypedReference Test1()   // CS1599  
   {  
      return null;  
   }  
  
   public ArgIterator Test2()   // CS1599  
   {  
      return null;  
   }  
}  
```
