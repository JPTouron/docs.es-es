# <a name="compiler-error-cs0202"></a><span data-ttu-id="f569a-103">Error del compilador CS0202</span><span class="sxs-lookup"><span data-stu-id="f569a-103">Compiler Error CS0202</span></span>

<span data-ttu-id="f569a-104">foreach requiere que el tipo de valor devuelto 'type' de 'type.GetEnumerator()' tenga un método MoveNext público y una propiedad Current pública adecuados</span><span class="sxs-lookup"><span data-stu-id="f569a-104">foreach requires that the return type 'type' of 'type.GetEnumerator()' must have a suitable public MoveNext method and public Current property</span></span>  
  
 <span data-ttu-id="f569a-105">Una función <xref:System.Collections.IEnumerable.GetEnumerator%2A> , que se usa para habilitar el uso de instrucciones foreach, no puede devolver un puntero o una matriz; debe devolver una instancia de clase que puede actuar como enumerador.</span><span class="sxs-lookup"><span data-stu-id="f569a-105">A <xref:System.Collections.IEnumerable.GetEnumerator%2A> function, used to enable the use of foreach statements, cannot return a pointer or array; it must return an instance of a class that is able to act as an enumerator.</span></span> <span data-ttu-id="f569a-106">Los requisitos apropiados para actuar como enumerador incluyen una propiedad Current pública y un método MoveNext público.</span><span class="sxs-lookup"><span data-stu-id="f569a-106">The proper requirements to serve as an enumerator include a public Current property and a public MoveNext method.</span></span>  
  
> [!NOTE]
> <span data-ttu-id="f569a-107">En C# 2.0, el compilador generará automáticamente Current y MoveNext.</span><span class="sxs-lookup"><span data-stu-id="f569a-107">In C# 2.0, the compiler will automatically generate Current and MoveNext for you.</span></span> <span data-ttu-id="f569a-108">Para obtener más información, vea el ejemplo de código en [interfaces genéricas](../programming-guide/generics/generic-interfaces.md).</span><span class="sxs-lookup"><span data-stu-id="f569a-108">For more information, see the code example in [Generic Interfaces](../programming-guide/generics/generic-interfaces.md).</span></span>  
  
 <span data-ttu-id="f569a-109">El ejemplo siguiente genera la advertencia CS0202:</span><span class="sxs-lookup"><span data-stu-id="f569a-109">The following sample generates CS0202:</span></span>  
  
```csharp  
// CS0202.cs  
  
public class C1  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public static implicit operator C1 (int c1)  
   {  
      return 0;  
   }  
}  
  
public class C2  
{  
   public int Current  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public bool MoveNext ()  
   {  
      return false;  
   }  
  
   public C1[] GetEnumerator ()  
   // try the following line instead  
   // public C1 GetEnumerator ()  
   {  
      return null;  
   }  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
      C2 c2 = new C2();  
  
      foreach (C1 x in c2)   // CS0202  
      {  
         System.Console.WriteLine(x.Current);  
      }  
   }  
}  
```
