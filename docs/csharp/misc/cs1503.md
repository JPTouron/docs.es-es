# <a name="compiler-error-cs1503"></a>Error del compilador CS1503
El argumento 'number' no se puede convertir de TypeA a TypeB  
  
 El tipo de un argumento de un método no coincide con el tipo que se pasó cuando se creó una instancia de la clase. Este error suele aparecer junto con CS1502. Consulte [CS1502](../language-reference/compiler-messages/cs1502.md) para ver una explicación de cómo solucionar este error.
