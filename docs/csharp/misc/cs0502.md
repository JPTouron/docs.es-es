 <span data-ttu-id="95cd7-105">Un miembro de clase no puede ser [abstract](../language-reference/keywords/abstract.md) y [sealed](../language-reference/keywords/sealed.md).</span><span class="sxs-lookup"><span data-stu-id="95cd7-105">A class member cannot be both [abstract](../language-reference/keywords/abstract.md) and [sealed](../language-reference/keywords/sealed.md).</span></span>

```csharp

{  
   abstract public void F();  
}

{  
   abstract sealed override public void F()   // CS0502

   }

{  
    public static void Main()  
    { }  
}  
```
