# <a name="compiler-warning-level-2-cs3021"></a><span data-ttu-id="a718f-102">Advertencia del compilador (nivel 2) CS3021</span><span class="sxs-lookup"><span data-stu-id="a718f-102">Compiler Warning (level 2) CS3021</span></span>

<span data-ttu-id="a718f-103">'tipo' no necesita ningún atributo CLSCompliant porque el ensamblado no tiene ningún atributo CLSCompliant.</span><span class="sxs-lookup"><span data-stu-id="a718f-103">'type' does not need a CLSCompliant attribute because the assembly does not have a CLSCompliant attribute</span></span>  
  
 <span data-ttu-id="a718f-104">Esta advertencia se produce si `[CLSCompliant(false)]` aparece en una clase de un ensamblado que no tiene un atributo de nivel de ensamblado CLSCompliant establecido en true (es decir, la línea `[assembly: CLSCompliant(true)]` ).</span><span class="sxs-lookup"><span data-stu-id="a718f-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="a718f-105">Dado que el ensamblado no se declara a sí mismo conforme a CLS, no hay ninguna necesidad de que nada dentro del ensamblado que se declare no conforme, ya que se supone que no es conforme.</span><span class="sxs-lookup"><span data-stu-id="a718f-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="a718f-106">Para obtener más información sobre la conformidad con CLS, consulte [independencia del lenguaje y componentes independientes del lenguaje](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="a718f-106">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
 <span data-ttu-id="a718f-107">Para hacer desaparecer esta advertencia, quite el atributo o agregue el atributo de nivel de ensamblado.</span><span class="sxs-lookup"><span data-stu-id="a718f-107">To get rid of this warning, remove the attribute or add the assembly level attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a718f-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="a718f-108">Example</span></span>  

 <span data-ttu-id="a718f-109">El ejemplo siguiente genera el error CS3021:</span><span class="sxs-lookup"><span data-stu-id="a718f-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
