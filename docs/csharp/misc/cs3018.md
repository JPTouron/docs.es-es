## <a name="example"></a><span data-ttu-id="59817-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="59817-108">Example</span></span>  

 <span data-ttu-id="59817-109">El ejemplo siguiente genera la advertencia CS3018.</span><span class="sxs-lookup"><span data-stu-id="59817-109">The following sample generates CS3018.</span></span>  
  
```csharp  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
