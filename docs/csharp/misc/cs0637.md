# <a name="compiler-error-cs0637"></a><span data-ttu-id="e9335-103">Error del compilador CS0637</span><span class="sxs-lookup"><span data-stu-id="e9335-103">Compiler Error CS0637</span></span>

<span data-ttu-id="e9335-104">El atributo FieldOffset no se permite en campos static o const.</span><span class="sxs-lookup"><span data-stu-id="e9335-104">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="e9335-105">El atributo [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) no se puede usar en campos marcados con [static](../language-reference/keywords/static.md) o [const](../language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="e9335-105">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../language-reference/keywords/static.md) or [const](../language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="e9335-106">El ejemplo siguiente genera la advertencia CS0637:</span><span class="sxs-lookup"><span data-stu-id="e9335-106">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
