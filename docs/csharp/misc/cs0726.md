# <a name="compiler-error-cs0726"></a>Error del compilador CS0726
'format specifier' no es un especificador de formato válido  
  
 Este error se produce en el depurador. Cuando escriba un nombre de variable en una de las ventanas del depurador, puede agregarle una coma y, a continuación, un especificador de formato. Algunos ejemplos de ello son `myInt, h` y `myString,nq`. Este error se produce cuando el compilador no reconoce los [especificadores de formato en C#](/visualstudio/debugger/format-specifiers-in-csharp).
