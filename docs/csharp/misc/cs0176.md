# <a name="compiler-error-cs0176"></a><span data-ttu-id="8bf2c-103">Error del compilador CS0176</span><span class="sxs-lookup"><span data-stu-id="8bf2c-103">Compiler Error CS0176</span></span>

<span data-ttu-id="8bf2c-104">No se puede acceder al miembro estático 'miembro' con una referencia de instancia; califíquelo con un nombre de tipo en su lugar.</span><span class="sxs-lookup"><span data-stu-id="8bf2c-104">Static member 'member' cannot be accessed with an instance reference; qualify it with a type name instead</span></span>  
  
 <span data-ttu-id="8bf2c-105">Solo puede usarse un nombre de clase para calificar una variable [estática](../language-reference/keywords/static.md) ; un nombre de instancia no puede ser un calificador.</span><span class="sxs-lookup"><span data-stu-id="8bf2c-105">Only a class name can be used to qualify a [static](../language-reference/keywords/static.md) variable; an instance name cannot be a qualifier.</span></span> <span data-ttu-id="8bf2c-106">Para más información, vea [Clases estáticas y sus miembros](../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span><span class="sxs-lookup"><span data-stu-id="8bf2c-106">For more information, see [Static Classes and Static Class Members](../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span></span>  
  
 <span data-ttu-id="8bf2c-107">El ejemplo siguiente genera la advertencia CS0176:</span><span class="sxs-lookup"><span data-stu-id="8bf2c-107">The following sample generates CS0176:</span></span>  
  
```csharp  
// CS0176.cs  
public class MyClass2  
{  
    public static int num;  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        MyClass2 mc2 = new MyClass2();  
        int i = mc2.num;   // CS0176  
        // try the following line instead  
        // int i = MyClass2.num;  
    }  
}  
```
