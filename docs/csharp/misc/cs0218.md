# <a name="compiler-error-cs0218"></a><span data-ttu-id="a7d4a-103">Error del compilador CS0218</span><span class="sxs-lookup"><span data-stu-id="a7d4a-103">Compiler Error CS0218</span></span>
<span data-ttu-id="a7d4a-104">El tipo ('type') debe incluir declaraciones de operador true y operador false</span><span class="sxs-lookup"><span data-stu-id="a7d4a-104">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="a7d4a-105">Si un tipo definido por el usuario sobrecarga el operador [&](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) o el [ operador&#124;](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), también debe definir operadores [true y false](../language-reference/operators/true-false-operators.md) , con el fin de hacer que el operador de [&& ](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) de cortocircuito o el [ operador de&#124;&#124; ](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) estén definidos.</span><span class="sxs-lookup"><span data-stu-id="a7d4a-105">If a user-defined type overloads the [& operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) or [&#124; operator](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), it must also define [true and false](../language-reference/operators/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) or [&#124;&#124; operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) defined.</span></span>
  
 <span data-ttu-id="a7d4a-106">El ejemplo siguiente genera la advertencia CS0218:</span><span class="sxs-lookup"><span data-stu-id="a7d4a-106">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
