 `using (f)`  
  
 `{`  
  
 `f = new thisType();`  
  
 `...`

## <a name="example"></a><span data-ttu-id="258a6-111">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="258a6-111">Example</span></span>  
 <span data-ttu-id="258a6-112">El código siguiente generará la advertencia CS0728.</span><span class="sxs-lookup"><span data-stu-id="258a6-112">The following code will generate warning CS0728.</span></span>  
  
```csharp

public class ValidBase : IDisposable  
{  
    public void Dispose() {  }  
}  
  
public class Logger  
{  
    public static void dummy()  
    {  
        ValidBase vb = null;  
        using (vb)
        {  
            vb = null;  // CS0728  
        }  
        vb = null;  
    }  
    public static void Main() { }  
}  
```
