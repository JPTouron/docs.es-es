# <a name="compiler-error-cs1686"></a><span data-ttu-id="c0310-102">Error del compilador CS1686</span><span class="sxs-lookup"><span data-stu-id="c0310-102">Compiler Error CS1686</span></span>

<span data-ttu-id="c0310-103">No puede tomarse la dirección de la variable local 'variable' o sus miembros y usarse en un método anónimo o una expresión lambda.</span><span class="sxs-lookup"><span data-stu-id="c0310-103">Local 'variable' or its members cannot have their address taken and be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="c0310-104">Este error se genera cuando se usa una variable, se intenta tomar su dirección y se realiza una de estas acciones dentro de un método anónimo.</span><span class="sxs-lookup"><span data-stu-id="c0310-104">This error is generated when you use a variable, and attempt to take its address, and one of these actions is done inside an anonymous method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c0310-105">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="c0310-105">Example</span></span>  

 <span data-ttu-id="c0310-106">El ejemplo siguiente genera la advertencia CS1686.</span><span class="sxs-lookup"><span data-stu-id="c0310-106">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
