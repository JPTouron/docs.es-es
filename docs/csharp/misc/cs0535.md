 <span data-ttu-id="90813-105">Se derivó una [clase](../language-reference/keywords/class.md) de una [interfaz](../language-reference/keywords/interface.md), pero la clase no implementó uno o varios de los miembros de la interfaz.</span><span class="sxs-lookup"><span data-stu-id="90813-105">A [class](../language-reference/keywords/class.md) derived from an [interface](../language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="90813-106">Una clase debe implementar todos los miembros de las interfaces de la que deriva, o bien se la debe declarar como `abstract`.</span><span class="sxs-lookup"><span data-stu-id="90813-106">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="90813-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="90813-107">Example</span></span>  

 <span data-ttu-id="90813-108">El ejemplo siguiente genera la advertencia CS0535.</span><span class="sxs-lookup"><span data-stu-id="90813-108">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
