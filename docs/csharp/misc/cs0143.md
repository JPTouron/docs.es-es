# <a name="compiler-error-cs0143"></a>Error del compilador CS0143

El tipo 'class' no tiene constructores definidos  
  
 No hay disponible un constructor adecuado. Este es el caso de los tipos de valor num√©rico integrado, que se inicializan asignando un valor a ellos.  
  
 El ejemplo siguiente genera la advertencia CS0143:  
  
```csharp  
// CS0143.cs  
class MyClass  
{  
   static public void Main ()  
   {  
      double d = new double(4.5);   // CS0143  
      // Try this line instead:  
      // double d = 4.5;  
   }  
}  
```
