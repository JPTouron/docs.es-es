## <a name="example"></a><span data-ttu-id="e5d6e-105">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="e5d6e-105">Example</span></span>  

 <span data-ttu-id="e5d6e-106">El ejemplo siguiente genera la advertencia CS1909.</span><span class="sxs-lookup"><span data-stu-id="e5d6e-106">The following sample generates CS1909.</span></span>  
  
```csharp

using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909

}  
```
