## <a name="example"></a><span data-ttu-id="13a1a-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="13a1a-106">Example</span></span>  

 <span data-ttu-id="13a1a-107">El ejemplo siguiente genera el error CS3002:</span><span class="sxs-lookup"><span data-stu-id="13a1a-107">The following example generates CS3002:</span></span>  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
