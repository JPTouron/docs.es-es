 <span data-ttu-id="b5565-105">Se intentó implementar un miembro de interfaz en una [clase](../language-reference/keywords/class.md) que no se deriva de la [interfaz](../language-reference/keywords/interface.md).</span><span class="sxs-lookup"><span data-stu-id="b5565-105">You attempted to implement an interface member in a [class](../language-reference/keywords/class.md) that does not derive from the [interface](../language-reference/keywords/interface.md).</span></span> <span data-ttu-id="b5565-106">Debe eliminar la implementación del miembro de interfaz o agregar la interfaz a la lista de clases base de la clase.</span><span class="sxs-lookup"><span data-stu-id="b5565-106">You should either delete the implementation of the interface member or add the interface to the base-class list of the class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b5565-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="b5565-107">Example</span></span>  

 <span data-ttu-id="b5565-108">El ejemplo siguiente genera la advertencia CS0540.</span><span class="sxs-lookup"><span data-stu-id="b5565-108">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
