## <a name="to-correct-this-error"></a><span data-ttu-id="b10de-109">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="b10de-109">To correct this error</span></span>

    ```csharp  
    return new Nullable<T>(t);  
    ```  
  
## <a name="example"></a><span data-ttu-id="b10de-112">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="b10de-112">Example</span></span>

<span data-ttu-id="b10de-113">El código siguiente genera el error CS0313:</span><span class="sxs-lookup"><span data-stu-id="b10de-113">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
