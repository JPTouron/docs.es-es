## <a name="example"></a><span data-ttu-id="a88f4-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="a88f4-107">Example</span></span>  
 <span data-ttu-id="a88f4-108">El ejemplo siguiente genera la advertencia CS0460.</span><span class="sxs-lookup"><span data-stu-id="a88f4-108">The following sample generates CS0460.</span></span>  
  
```csharp  
// CS0460.cs  
// compile with: /target:library  
class BaseClass
{  
   BaseClass() { }  
}  
  
interface I  
{  
   void F1<T>() where T : BaseClass;  
   void F2<T>() where T : struct;  
   void F3<T>() where T : BaseClass;  
}  
  
class ExpImpl : I  
{  
   void I.F1<T>() where T : BaseClass {}   // CS0460  
   void I.F2<T>() where T : class {}  // CS0460  
}  
```
