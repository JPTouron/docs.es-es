 <span data-ttu-id="43e35-105">Se repitió una etiqueta en una instrucción [switch](../language-reference/keywords/switch.md) .</span><span class="sxs-lookup"><span data-stu-id="43e35-105">A label was repeated in a [switch](../language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="43e35-106">Para obtener más información, consulta [switch](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="43e35-106">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="43e35-107">El ejemplo siguiente genera la advertencia CS0152:</span><span class="sxs-lookup"><span data-stu-id="43e35-107">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
