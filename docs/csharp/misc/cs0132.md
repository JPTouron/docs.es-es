 <span data-ttu-id="0c2ef-107">El ejemplo siguiente genera la advertencia CS0132:</span><span class="sxs-lookup"><span data-stu-id="0c2ef-107">The following sample generates CS0132:</span></span>  
  
```csharp  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
