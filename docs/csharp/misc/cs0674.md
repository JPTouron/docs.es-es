 <span data-ttu-id="37b3b-106">El compilador de C# no permite el uso de <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) en su lugar.</span><span class="sxs-lookup"><span data-stu-id="37b3b-106">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>

```csharp

{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead

         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
