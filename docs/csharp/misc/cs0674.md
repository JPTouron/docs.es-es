<span data-ttu-id="33d2d-104">No use 'System.ParamArrayAttribute'.</span><span class="sxs-lookup"><span data-stu-id="33d2d-104">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="33d2d-105">Use la palabra clave 'params' en su lugar.</span><span class="sxs-lookup"><span data-stu-id="33d2d-105">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="33d2d-106">El compilador de C# no permite el uso de <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) en su lugar.</span><span class="sxs-lookup"><span data-stu-id="33d2d-106">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="33d2d-107">El ejemplo siguiente genera la advertencia CS0674:</span><span class="sxs-lookup"><span data-stu-id="33d2d-107">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
