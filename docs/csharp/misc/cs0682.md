# <a name="compiler-error-cs0682"></a>Error del compilador CS0682

'type1' no puede implementar 'type2' porque no es compatible con el lenguaje  
  
 Este error se produce cuando se intenta implementar una interfaz escrita en otro lenguaje y el compilador no admite la interfaz.
