# <a name="compiler-error-cs0027"></a><span data-ttu-id="e375f-103">Error del compilador CS0027</span><span class="sxs-lookup"><span data-stu-id="e375f-103">Compiler Error CS0027</span></span>
<span data-ttu-id="e375f-104">La palabra clave 'this' no está disponible en el contexto actual</span><span class="sxs-lookup"><span data-stu-id="e375f-104">Keyword 'this' is not available in the current context</span></span>  
  
 <span data-ttu-id="e375f-105">La palabra clave [this](../language-reference/keywords/this.md) se encuentra fuera de una propiedad, un método o un constructor.</span><span class="sxs-lookup"><span data-stu-id="e375f-105">The [this](../language-reference/keywords/this.md) keyword was found outside of a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="e375f-106">Para corregir este error, modifique la instrucción para eliminar el uso de la palabra clave `this` y/o mueva parte de la instrucción, o su totalidad, dentro de una propiedad, un método o un constructor.</span><span class="sxs-lookup"><span data-stu-id="e375f-106">To fix this error, either modify the statement to eliminate use of the `this` keyword, and/or move part or all of the statement inside a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="e375f-107">El ejemplo siguiente genera el error CS0027:</span><span class="sxs-lookup"><span data-stu-id="e375f-107">The following example generates CS0027:</span></span>  
  
```csharp  
using System;  
using System.Collections.Generic;  
using System.Text;  
  
namespace ConsoleApplication3  
{  
    class MyClass  
    {  
  
        int err1 = this.Fun() + 1;  // CS0027
  
        public int Fun()  
        {  
            return 10;  
        }  
  
        public void Test()  
        {  
            // valid use of this  
            int err = this.Fun() + 1;  
            Console.WriteLine(err);  
        }  
  
        public static void Main()  
        {  
            MyClass c = new MyClass();  
            c.Test();  
        }  
    }  
}  
```
