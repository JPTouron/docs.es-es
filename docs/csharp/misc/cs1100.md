 <span data-ttu-id="ef498-105">El modificador `this` solo se permite en el primer parámetro de un método, e indica al compilador que el método es un método de extensión.</span><span class="sxs-lookup"><span data-stu-id="ef498-105">The `this` modifier is allowed only on the first parameter of a method, which indicates to the compiler that the method is an extension method.</span></span>
