# <a name="compiler-error-cs1563"></a>Error del compilador CS1563

El resultado 'element' no tiene archivos de código fuente  
  
 Se especificó la opción del compilador [-out](../language-reference/compiler-options/out-compiler-option.md) , pero no siguen los archivos de código fuente. Debe o bien no utilizar **/out**, o bien especificar los archivos de código fuente después de **/out**.
