# <a name="compiler-error-cs1618"></a><span data-ttu-id="1e1b1-103">Error del compilador CS1618</span><span class="sxs-lookup"><span data-stu-id="1e1b1-103">Compiler Error CS1618</span></span>

<span data-ttu-id="1e1b1-104">No se puede crear un delegado con 'method' porque tiene un atributo Conditional</span><span class="sxs-lookup"><span data-stu-id="1e1b1-104">Cannot create delegate with 'method' because it has a Conditional attribute</span></span>  
  
 <span data-ttu-id="1e1b1-105">No se puede crear un delegado con un método condicional porque el método puede que no exista en algunas compilaciones.</span><span class="sxs-lookup"><span data-stu-id="1e1b1-105">You cannot create a delegate with a conditional method because the method might not exist in some builds.</span></span>  
  
 <span data-ttu-id="1e1b1-106">El ejemplo siguiente genera la advertencia CS1618:</span><span class="sxs-lookup"><span data-stu-id="1e1b1-106">The following sample generates CS1618:</span></span>  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
