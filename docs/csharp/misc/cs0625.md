# <a name="compiler-error-cs0625"></a><span data-ttu-id="169fe-103">Error del compilador CS0625</span><span class="sxs-lookup"><span data-stu-id="169fe-103">Compiler Error CS0625</span></span>

<span data-ttu-id="169fe-104">' campo ': los tipos de campo de instancia marcados con StructLayout (LayoutKind. Explicit) deben tener un atributo FieldOffset.</span><span class="sxs-lookup"><span data-stu-id="169fe-104">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="169fe-105">Cuando se marca una estructura con un atributo **StructLayout** explícito, todos los campos de la estructura deben tener el atributo [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) .</span><span class="sxs-lookup"><span data-stu-id="169fe-105">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="169fe-106">Para obtener más información, consulte [Clase StructLayoutAttribute](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span><span class="sxs-lookup"><span data-stu-id="169fe-106">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span>

<span data-ttu-id="169fe-107">El ejemplo siguiente genera la advertencia CS0625:</span><span class="sxs-lookup"><span data-stu-id="169fe-107">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
