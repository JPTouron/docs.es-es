# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="5c4c7-103">Advertencia del compilador (nivel 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="5c4c7-103">Compiler Warning (level 1) CS0183</span></span>

<span data-ttu-id="5c4c7-104">La expresión dada siempre es del tipo facilitado ('type')</span><span class="sxs-lookup"><span data-stu-id="5c4c7-104">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="5c4c7-105">Si una instrucción condicional siempre se evalúa como **true**, no necesita una instrucción condicional.</span><span class="sxs-lookup"><span data-stu-id="5c4c7-105">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="5c4c7-106">Esta advertencia se produce cuando intenta evaluar un tipo con el operador **is** .</span><span class="sxs-lookup"><span data-stu-id="5c4c7-106">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="5c4c7-107">Si la evaluación es un tipo de valor, la comprobación es innecesaria.</span><span class="sxs-lookup"><span data-stu-id="5c4c7-107">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="5c4c7-108">El ejemplo siguiente genera la advertencia CS0183:</span><span class="sxs-lookup"><span data-stu-id="5c4c7-108">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);
   }  
}  
```
