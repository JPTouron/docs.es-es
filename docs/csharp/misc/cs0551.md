## <a name="example"></a><span data-ttu-id="fe733-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="fe733-107">Example</span></span>  
 <span data-ttu-id="fe733-108">El ejemplo siguiente genera la advertencia CS0551.</span><span class="sxs-lookup"><span data-stu-id="fe733-108">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
