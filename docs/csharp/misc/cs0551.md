 <span data-ttu-id="540bf-105">Una clase que implementa explícitamente la propiedad de la interfaz debe implementar todos los descriptores de acceso que define la interfaz.</span><span class="sxs-lookup"><span data-stu-id="540bf-105">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>

## <a name="example"></a><span data-ttu-id="540bf-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="540bf-107">Example</span></span>  

 <span data-ttu-id="540bf-108">El ejemplo siguiente genera la advertencia CS0551.</span><span class="sxs-lookup"><span data-stu-id="540bf-108">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
