# <a name="compiler-warning-level-1-cs1592"></a>Advertencia del compilador (nivel 1) CS1592

XML con formato incorrecto en el archivo de comentarios de inclusión: 'reason'  
  
 Se encontró un problema, que se indica como ***motivo***, en el archivo especificado por la [\<include>](../programming-guide/xmldoc/include.md) etiqueta.
