<span data-ttu-id="7573e-104">Referencia ambigua en el atributo cref: 'Method Name1'.</span><span class="sxs-lookup"><span data-stu-id="7573e-104">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="7573e-105">Se asume 'Method Name2', pero podría también coincidir con otras sobrecargas que incluyan 'Method Name3'.</span><span class="sxs-lookup"><span data-stu-id="7573e-105">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="7573e-106">En un comentario de documentación XML en el código, no se pudo resolver una referencia.</span><span class="sxs-lookup"><span data-stu-id="7573e-106">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="7573e-107">Esto puede ocurrir si el método está sobrecargado, o si se encuentran dos identificadores diferentes con el mismo nombre.</span><span class="sxs-lookup"><span data-stu-id="7573e-107">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="7573e-108">Para resolver la advertencia, use un nombre completo para eliminar la ambigüedad de la referencia, o incluya la sobrecarga concreta entre paréntesis.</span><span class="sxs-lookup"><span data-stu-id="7573e-108">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="7573e-109">El ejemplo siguiente genera la advertencia CS0419.</span><span class="sxs-lookup"><span data-stu-id="7573e-109">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
