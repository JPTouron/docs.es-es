# <a name="compiler-error-cs1566"></a>Error del compilador CS1566

Error al leer el recurso incrustado 'file': 'reason'  
  
 El compilador tenía problemas con el nombre de archivo pasado a la opción del compilador [-Resource](../language-reference/compiler-options/resource-compiler-option.md) .
