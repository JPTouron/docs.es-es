# <a name="compiler-error-cs1037"></a>Error del compilador CS1037

Se esperaba un operador sobrecargable  
  
 Al especificar un comentario con [-doc](../language-reference/compiler-options/doc-compiler-option.md), el compilador encontró un vínculo no válido.
