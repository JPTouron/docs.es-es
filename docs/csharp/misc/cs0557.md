# <a name="compiler-error-cs0557"></a><span data-ttu-id="8005d-103">Error del compilador CS0557</span><span class="sxs-lookup"><span data-stu-id="8005d-103">Compiler Error CS0557</span></span>
<span data-ttu-id="8005d-104">Conversión definida por el usuario duplicada en el tipo 'class'</span><span class="sxs-lookup"><span data-stu-id="8005d-104">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="8005d-105">No se permiten rutinas de conversión duplicadas en una clase.</span><span class="sxs-lookup"><span data-stu-id="8005d-105">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="8005d-106">El siguiente ejemplo genera el error CS0557:</span><span class="sxs-lookup"><span data-stu-id="8005d-106">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
