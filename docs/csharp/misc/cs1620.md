# <a name="compiler-error-cs1620"></a><span data-ttu-id="3c6ea-103">Error del compilador CS1620</span><span class="sxs-lookup"><span data-stu-id="3c6ea-103">Compiler Error CS1620</span></span>

<span data-ttu-id="3c6ea-104">Se debe pasar el argumento 'number' con la palabra clave 'keyword'.</span><span class="sxs-lookup"><span data-stu-id="3c6ea-104">Argument 'number' must be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="3c6ea-105">Este error se produce si se pasa un argumento a una función que toma un parámetro [ref](../language-reference/keywords/ref.md) o [out](../language-reference/keywords/out-parameter-modifier.md) y no incluye la palabra clave `ref` o `out` en el momento de la llamada, o bien incluye la palabra clave incorrecta.</span><span class="sxs-lookup"><span data-stu-id="3c6ea-105">This error occurs if you are passing an argument to a function that takes a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameter and you don't include the `ref` or `out` keyword at the point of call, or you include the wrong keyword.</span></span> <span data-ttu-id="3c6ea-106">El texto del error indica la palabra clave adecuada para usarse y el argumento que produjo el error.</span><span class="sxs-lookup"><span data-stu-id="3c6ea-106">The error text indicates the appropriate keyword to use and which argument caused the failure.</span></span>  
  
 <span data-ttu-id="3c6ea-107">El ejemplo siguiente genera la advertencia CS1620:</span><span class="sxs-lookup"><span data-stu-id="3c6ea-107">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
