<span data-ttu-id="7d6b0-104">'class': una clase con el atributo ComImport no puede especificar ninguna clase base</span><span class="sxs-lookup"><span data-stu-id="7d6b0-104">'class': a class with the ComImport attribute cannot specify a base class</span></span>

 <span data-ttu-id="7d6b0-107">El ejemplo siguiente genera la advertencia CS0424:</span><span class="sxs-lookup"><span data-stu-id="7d6b0-107">The following sample generates CS0424:</span></span>

