# <a name="compiler-error-cs1678"></a><span data-ttu-id="f490c-102">Error del compilador CS1678</span><span class="sxs-lookup"><span data-stu-id="f490c-102">Compiler Error CS1678</span></span>

<span data-ttu-id="f490c-103">El parámetro 'número' se declaró como tipo 'tipo1', pero debería ser 'tipo2'</span><span class="sxs-lookup"><span data-stu-id="f490c-103">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="f490c-104">Este error se produce cuando el tipo de parámetro de un método anónimo no coincide con la declaración del delegado al que se va a convertir el método.</span><span class="sxs-lookup"><span data-stu-id="f490c-104">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="f490c-105">El ejemplo siguiente genera la advertencia CS1678:</span><span class="sxs-lookup"><span data-stu-id="f490c-105">The following sample generates CS1678:</span></span>  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors
{  
   static void Main()
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
