# <a name="compiler-error-cs0601"></a><span data-ttu-id="972c5-103">Error del compilador CS0601</span><span class="sxs-lookup"><span data-stu-id="972c5-103">Compiler Error CS0601</span></span>
<span data-ttu-id="972c5-104">El atributo DllImport se debe especificar en un método marcado como 'static' y 'extern'</span><span class="sxs-lookup"><span data-stu-id="972c5-104">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="972c5-105">El atributo `DllImport` se usó en un método que no tenía las palabras clave de acceso correctas.</span><span class="sxs-lookup"><span data-stu-id="972c5-105">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="972c5-106">El ejemplo siguiente genera la advertencia CS0601:</span><span class="sxs-lookup"><span data-stu-id="972c5-106">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
