 <span data-ttu-id="89f13-105">Una construcción [pública](../language-reference/keywords/public.md) debe devolver un objeto accesible públicamente.</span><span class="sxs-lookup"><span data-stu-id="89f13-105">A [public](../language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="89f13-108">El ejemplo siguiente genera la advertencia CS0061.</span><span class="sxs-lookup"><span data-stu-id="89f13-108">The following sample generates CS0061.</span></span>

```csharp  
public interface AA : A {}  // CS0061  
  
// OK

internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
