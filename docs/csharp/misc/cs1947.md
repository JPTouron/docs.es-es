## <a name="example"></a><span data-ttu-id="53472-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="53472-109">Example</span></span>

```csharp  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };

                let k = i

    }  
}  
```
