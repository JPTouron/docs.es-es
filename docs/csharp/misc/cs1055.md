# <a name="compiler-error-cs1055"></a>Error del compilador CS1055
Se esperaba un descriptor de acceso add o remove  
  
 Si su [evento](../language-reference/keywords/event.md) no se ha declarado como campo, debe definir las funciones de descriptor de acceso **add** y **remove** .  
  
 El ejemplo siguiente genera la advertencia CS1055:  
  
```csharp  
// CS1055.cs  
delegate void del();  
class Test  
{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
