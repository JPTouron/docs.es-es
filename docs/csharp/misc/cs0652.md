 <span data-ttu-id="3f119-105">El compilador detectó una comparación entre una constante y una variable donde la constante está fuera del rango de la variable.</span><span class="sxs-lookup"><span data-stu-id="3f119-105">The compiler detected a comparison between a constant and a variable where the constant is out of the range of the variable.</span></span>  
  
 <span data-ttu-id="3f119-106">El ejemplo siguiente genera la advertencia CS0652:</span><span class="sxs-lookup"><span data-stu-id="3f119-106">The following sample generates CS0652:</span></span>

{  
   private static byte i = 0;  
   public static void Main()

      short j = 256;  
      if (i == 256)   // CS0652, 256 is out of range for byte

   }  
}
