# <a name="compiler-error-cs0569"></a>Error del compilador CS0569
'method2': no se puede reemplazar 'method1' porque el lenguaje no lo admite  
  
 Este error se produce al derivar de una clase base que se escribió en otro lenguaje y cuando el compilador no admite el método que intenta reemplazar.
