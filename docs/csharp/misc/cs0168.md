<span data-ttu-id="2ce78-106">En el ejemplo siguiente se genera una advertencia CS0168:</span><span class="sxs-lookup"><span data-stu-id="2ce78-106">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
