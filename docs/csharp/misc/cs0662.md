# <a name="compiler-error-cs0662"></a><span data-ttu-id="04ad3-103">Error del compilador CS0662</span><span class="sxs-lookup"><span data-stu-id="04ad3-103">Compiler Error CS0662</span></span>

<span data-ttu-id="04ad3-104">'method' no puede especificar solo el atributo Out en un parámetro ref.</span><span class="sxs-lookup"><span data-stu-id="04ad3-104">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="04ad3-105">Use ambos atributos In y Out, o bien ninguno.</span><span class="sxs-lookup"><span data-stu-id="04ad3-105">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="04ad3-106">Un método de interfaz tiene un parámetro que usa [ref](../language-reference/keywords/ref.md) con el atributo [Out](xref:System.Runtime.InteropServices.OutAttribute) solamente.</span><span class="sxs-lookup"><span data-stu-id="04ad3-106">An interface method has a parameter that uses [ref](../language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="04ad3-107">Un parámetro `ref` que usa el atributo **Out** también debe usar el atributo [In](xref:System.Runtime.InteropServices.InAttribute) .</span><span class="sxs-lookup"><span data-stu-id="04ad3-107">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="04ad3-108">El ejemplo siguiente genera la advertencia CS0662:</span><span class="sxs-lookup"><span data-stu-id="04ad3-108">The following sample generates CS0662:</span></span>  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
