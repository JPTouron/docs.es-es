# <a name="compiler-error-cs0159"></a><span data-ttu-id="91ab3-103">Error del compilador CS0159</span><span class="sxs-lookup"><span data-stu-id="91ab3-103">Compiler Error CS0159</span></span>
<span data-ttu-id="91ab3-104">No existe la etiqueta 'label' en el ámbito de la instrucción goto</span><span class="sxs-lookup"><span data-stu-id="91ab3-104">No such label 'label' within the scope of the goto statement</span></span>  
  
 <span data-ttu-id="91ab3-105">La etiqueta a la que hace referencia la instrucción [goto](../language-reference/keywords/goto.md) no se encontró en el ámbito de la instrucción `goto` .</span><span class="sxs-lookup"><span data-stu-id="91ab3-105">The label referenced by the [goto](../language-reference/keywords/goto.md) statement could not be found within the scope of the `goto` statement.</span></span>  
  
 <span data-ttu-id="91ab3-106">El ejemplo siguiente genera la advertencia CS0159:</span><span class="sxs-lookup"><span data-stu-id="91ab3-106">The following sample generates CS0159:</span></span>  
  
```csharp  
// CS0159.cs  
public class Class1  
{  
   public static void Main()  
   {  
      int i = 0;  
  
      switch (i)  
      {  
         case 1:  
            goto case 3;   // CS0159, case 3 label does not exist  
         case 2:  
            break;  
      }  
      goto NOWHERE;   // CS0159, NOWHERE label does not exist  
   }  
}  
```
