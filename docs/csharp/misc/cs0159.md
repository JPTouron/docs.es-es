# <a name="compiler-error-cs0159"></a>Error del compilador CS0159
No existe la etiqueta 'label' en el ámbito de la instrucción goto  
  
 La etiqueta a la que hace referencia la instrucción [goto](../language-reference/keywords/goto.md) no se encontró en el ámbito de la instrucción `goto` .  
  
 El ejemplo siguiente genera la advertencia CS0159:  
  
```csharp  
// CS0159.cs  
public class Class1  
{  
   public static void Main()  
   {  
      int i = 0;  
  
      switch (i)  
      {  
         case 1:  
            goto case 3;   // CS0159, case 3 label does not exist  
         case 2:  
            break;  
      }  
      goto NOWHERE;   // CS0159, NOWHERE label does not exist  
   }  
}  
```
