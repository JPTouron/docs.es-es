# <a name="compiler-error-cs0542"></a><span data-ttu-id="a0fee-103">Error del compilador CS0542</span><span class="sxs-lookup"><span data-stu-id="a0fee-103">Compiler Error CS0542</span></span>
<span data-ttu-id="a0fee-104">'user-defined type': los nombres de los miembros no pueden coincidir con sus tipos envolventes</span><span class="sxs-lookup"><span data-stu-id="a0fee-104">'user-defined type' : member names cannot be the same as their enclosing type</span></span>  
  
 <span data-ttu-id="a0fee-105">Los miembros de una clase o una estructura no pueden tener el mismo nombre que la clase o la estructura, a menos que el miembro sea un constructor.</span><span class="sxs-lookup"><span data-stu-id="a0fee-105">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>  
  
 <span data-ttu-id="a0fee-106">El ejemplo siguiente genera la advertencia CS0542:</span><span class="sxs-lookup"><span data-stu-id="a0fee-106">The following sample generates CS0542:</span></span>  
  
```csharp

{  
    public int C;  
}  
```  
  
 <span data-ttu-id="a0fee-107">Este error podría producirse si coloca accidentalmente un tipo de valor devuelto en un constructor, que a todos los efectos se convierte en un método normal.</span><span class="sxs-lookup"><span data-stu-id="a0fee-107">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="a0fee-108">En el ejemplo siguiente se genera el error CS0542 porque `F` es un método, no un constructor, ya que tiene un tipo de valor devuelto:</span><span class="sxs-lookup"><span data-stu-id="a0fee-108">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>  
  
```csharp

{  
   // Remove void from F() to resolve the problem.  
   void F()   // CS0542, same name as the class  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
