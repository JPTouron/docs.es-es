# <a name="compiler-error-cs0750"></a><span data-ttu-id="4a76a-103">Error del compilador CS0750</span><span class="sxs-lookup"><span data-stu-id="4a76a-103">Compiler Error CS0750</span></span>

<span data-ttu-id="4a76a-104">Un método parcial no puede tener modificadores de acceso ni los modificadores virtual, abstract, override, new, sealed o extern.</span><span class="sxs-lookup"><span data-stu-id="4a76a-104">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="4a76a-105">Debido a su comportamiento especial, los métodos parciales están sujetos a restricciones en cuanto a los modificadores que pueden aceptar.</span><span class="sxs-lookup"><span data-stu-id="4a76a-105">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4a76a-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="4a76a-106">To correct this error</span></span>  
  
1. <span data-ttu-id="4a76a-107">Quite el modificador no autorizado de la declaración del método.</span><span class="sxs-lookup"><span data-stu-id="4a76a-107">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4a76a-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="4a76a-108">Example</span></span>  

 <span data-ttu-id="4a76a-109">El siguiente ejemplo genera el error CS0750:</span><span class="sxs-lookup"><span data-stu-id="4a76a-109">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
