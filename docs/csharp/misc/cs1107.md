## <a name="example"></a><span data-ttu-id="3df9d-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="3df9d-106">Example</span></span>

```csharp

    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107

    // Regular Instance Method
```
