# <a name="compiler-warning-level-1-cs1635"></a><span data-ttu-id="628bf-103">Advertencia del compilador (nivel 1) CS1635</span><span class="sxs-lookup"><span data-stu-id="628bf-103">Compiler Warning (level 1) CS1635</span></span>

<span data-ttu-id="628bf-104">No se puede restaurar la advertencia 'warning code' porque estaba deshabilitada globalmente.</span><span class="sxs-lookup"><span data-stu-id="628bf-104">Cannot restore warning 'warning code' because it was disabled globally</span></span>  
  
 <span data-ttu-id="628bf-105">Esta advertencia se produce si usa la opción de línea de comandos **/nowarn** o la configuración de proyecto para deshabilitar una advertencia en toda la unidad de compilación, pero usa `#pragma warning restore` para intentar restaurar dicha advertencia.</span><span class="sxs-lookup"><span data-stu-id="628bf-105">This warning occurs if you use the **/nowarn** command line option or project setting to disable a warning for the entire compilation unit, but you use `#pragma warning restore` to attempt to restore that warning.</span></span> <span data-ttu-id="628bf-106">Para resolver este error, quite la opción de línea de comandos /nowarn o la configuración de proyecto, o quite el `#pragma warning restore` de cualquier advertencia que esté deshabilitando a través de la línea de comandos o la configuración del proyecto.</span><span class="sxs-lookup"><span data-stu-id="628bf-106">To resolve this error, remove the /nowarn command line option or project setting, or remove the `#pragma warning restore` for any warnings you are disabling via the command line or project settings.</span></span> <span data-ttu-id="628bf-107">Para más información, vea el tema [#pragma warning](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) .</span><span class="sxs-lookup"><span data-stu-id="628bf-107">For more information, see the [#pragma warning](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) topic.</span></span>  
  
 <span data-ttu-id="628bf-108">El ejemplo siguiente genera la advertencia CS1635:</span><span class="sxs-lookup"><span data-stu-id="628bf-108">The following sample generates CS1635:</span></span>  
  
```csharp  
// CS1635.cs  
// compile with: /w:1 /nowarn:162  
  
enum MyEnum {one=1,two=2,three=3};  
  
class MyClass  
{  
    public static void Main()  
    {  
#pragma warning disable 162  
  
    if (MyEnum.three == MyEnum.two)  
        System.Console.WriteLine("Duplicate");  
  
#pragma warning restore 162  
    }  
}  
```
