<span data-ttu-id="93b9f-104">Dado que 'clase' tiene el atributo ComImport, 'método' debe ser extern o abstract.</span><span class="sxs-lookup"><span data-stu-id="93b9f-104">Since 'class' has the ComImport attribute, 'method' must be extern or abstract</span></span>

```csharp

  ComImport,

  {  
    ImageProperties i = new ImageProperties();  
  }

```
