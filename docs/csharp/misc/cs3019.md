## <a name="example"></a><span data-ttu-id="7ff67-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="7ff67-107">Example</span></span>  

 <span data-ttu-id="7ff67-108">El ejemplo siguiente genera la advertencia CS3019:</span><span class="sxs-lookup"><span data-stu-id="7ff67-108">The following sample generates CS3019:</span></span>  
  
```csharp  
// CS3019.cs  
// compile with: /W:2  
  
using System;  
  
[assembly: CLSCompliant(true)]  
  
// To fix the error, remove the next line  
[CLSCompliant(true)]  // CS3019

{  
    [CLSCompliant(false)]  // CS3019  
    void Foo()  
    {  
    }  
  
    static void Main()  
    {  
    }  
}  
```
