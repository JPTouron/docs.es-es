# <a name="compiler-error-cs0743"></a><span data-ttu-id="b62ba-103">Error del compilador CS0743</span><span class="sxs-lookup"><span data-stu-id="b62ba-103">Compiler Error CS0743</span></span>
<span data-ttu-id="b62ba-104">Se esperaba la palabra clave contextual 'on'</span><span class="sxs-lookup"><span data-stu-id="b62ba-104">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="b62ba-105">El patrón para una cláusula `join` es `join`...`in`...`on`...`equals`, como se muestra en este ejemplo:</span><span class="sxs-lookup"><span data-stu-id="b62ba-105">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

## <a name="example"></a><span data-ttu-id="b62ba-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="b62ba-108">Example</span></span>  
 <span data-ttu-id="b62ba-109">El código siguiente genera el error CS0743:</span><span class="sxs-lookup"><span data-stu-id="b62ba-109">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
