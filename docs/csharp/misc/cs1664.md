# <a name="compiler-error-cs1664"></a>Error del compilador CS1664

El búfer de tamaño fijo de longitud 'length' y tipo 'type' es demasiado grande  
  
 El tamaño máximo de un búfer de tamaño fijo (que determina el valor de la longitud multiplicada por el tamaño del elemento) es 2^31 = 268435455.
