# <a name="compiler-error-cs0526"></a>Error del compilador CS0526
Las interfaces no pueden incluir constructores  
  
 No se pueden definir constructores para [interfaces](../language-reference/keywords/interface.md). Un método se considera un constructor si tiene el mismo nombre que la clase y ningún tipo de valor devuelto.  
  
 El ejemplo siguiente genera la advertencia CS0526:  
  
```csharp  
// CS0526.cs  
namespace x  
{  
   public interface clx  
   {  
      public clx()   // CS0526  
      {  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
