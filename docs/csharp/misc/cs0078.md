# <a name="compiler-warning-level-4-cs0078"></a><span data-ttu-id="ddf9b-103">Advertencia del compilador (nivel 4) CS0078</span><span class="sxs-lookup"><span data-stu-id="ddf9b-103">Compiler Warning (level 4) CS0078</span></span>

<span data-ttu-id="ddf9b-104">El sufijo 'l' se confunde fácilmente con el dígito '1': utilice 'L' para mayor claridad</span><span class="sxs-lookup"><span data-stu-id="ddf9b-104">The 'l' suffix is easily confused with the digit '1' -- use 'L' for clarity</span></span>  
  
 <span data-ttu-id="ddf9b-105">El compilador advierte cuando detecta una conversión al tipo long mediante una letra l minúscula en lugar de una L mayúscula.</span><span class="sxs-lookup"><span data-stu-id="ddf9b-105">The compiler warns when it detects a cast to long using a lowercase l instead of an uppercase L.</span></span>  
  
 <span data-ttu-id="ddf9b-106">El ejemplo siguiente genera la advertencia CS0078:</span><span class="sxs-lookup"><span data-stu-id="ddf9b-106">The following sample generates CS0078:</span></span>  
  
```csharp  
// CS0078.cs  
// compile with: /W:4  
class test {  
   public static void TestL(long i)  
   {  
   }  
  
   public static void TestL(int i)  
   {  
   }  
  
   public static void Main()  
   {  
      TestL(25l);   // CS0078  
      // try the following line instead  
      // TestL(25L);  
   }  
}  
```
