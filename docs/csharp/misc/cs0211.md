# <a name="compiler-error-cs0211"></a>Error del compilador CS0211
No se puede adquirir la dirección de la expresión dada  
  
 Puede tomar la dirección de campos, variables locales y el direccionamiento indirecto de punteros, pero no puede tomar, por ejemplo, la dirección de la suma de dos variables locales. Para obtener más información, vea [Código no seguro y punteros (Guía de programación de C#)](../programming-guide/unsafe-code-pointers/index.md).  
  
 El ejemplo siguiente genera la advertencia CS0211:  
  
```csharp  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
