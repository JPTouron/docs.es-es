# <a name="compiler-error-cs0835"></a><span data-ttu-id="16602-103">Error del compilador CS0835</span><span class="sxs-lookup"><span data-stu-id="16602-103">Compiler Error CS0835</span></span>

<span data-ttu-id="16602-104">No se puede convertir una expresión lambda a un árbol de expresión cuyo argumento de tipo 'tipo' no sea un tipo delegado.</span><span class="sxs-lookup"><span data-stu-id="16602-104">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="16602-105">Si una expresión lambda se convierte a un árbol de expresión, dicho árbol debe tener un tipo delegado para su argumento.</span><span class="sxs-lookup"><span data-stu-id="16602-105">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="16602-106">Además, la expresión lambda debe poder convertirse al tipo delegado.</span><span class="sxs-lookup"><span data-stu-id="16602-106">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="16602-107">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="16602-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="16602-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="16602-109">Example</span></span>  

 <span data-ttu-id="16602-110">El ejemplo siguiente genera el error CS0835:</span><span class="sxs-lookup"><span data-stu-id="16602-110">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
