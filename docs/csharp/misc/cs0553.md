# <a name="compiler-error-cs0553"></a><span data-ttu-id="a665e-103">Error del compilador CS0553</span><span class="sxs-lookup"><span data-stu-id="a665e-103">Compiler Error CS0553</span></span>
<span data-ttu-id="a665e-104">'conversion routine': el usuario definió la conversión a/desde la clase base</span><span class="sxs-lookup"><span data-stu-id="a665e-104">'conversion routine' : user defined conversion to/from base class</span></span>  
  
 <span data-ttu-id="a665e-105">No se permiten las conversiones definidas por el usuario a valores de una clase base; no es necesario este tipo de operador.</span><span class="sxs-lookup"><span data-stu-id="a665e-105">User-defined conversions to values of a base class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="a665e-106">El ejemplo siguiente genera la advertencia CS0553:</span><span class="sxs-lookup"><span data-stu-id="a665e-106">The following sample generates CS0553:</span></span>  
  
```csharp  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553  
      public static implicit operator ii(a aa) // CS0553  
      {  
         return new ii();  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
