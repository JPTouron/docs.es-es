# <a name="compiler-error-cs0254"></a><span data-ttu-id="335ba-103">Error del compilador CS0254</span><span class="sxs-lookup"><span data-stu-id="335ba-103">Compiler Error CS0254</span></span>
<span data-ttu-id="335ba-104">El lado derecho de una asignación de instrucción fixed puede no ser una expresión de conversión</span><span class="sxs-lookup"><span data-stu-id="335ba-104">The right hand side of a fixed statement assignment may not be a cast expression</span></span>  
  
 <span data-ttu-id="335ba-105">Es posible que el lado derecho de una expresión [fixed](../language-reference/keywords/fixed-statement.md) no pueda usar una conversión.</span><span class="sxs-lookup"><span data-stu-id="335ba-105">The right side of a [fixed](../language-reference/keywords/fixed-statement.md) expression may not use a cast.</span></span> <span data-ttu-id="335ba-106">Para obtener más información, vea [Código no seguro y punteros (Guía de programación de C#)](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="335ba-106">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="335ba-107">El ejemplo siguiente genera la advertencia CS0254:</span><span class="sxs-lookup"><span data-stu-id="335ba-107">The following sample generates CS0254:</span></span>  
  
```csharp  
// CS0254.cs  
// compile with: /unsafe  
class Point  
{  
   public uint x, y;  
}  
  
class FixedTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      *p *= *p;  
   }  
  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
      pt.x = 5;  
      pt.y = 6;  
  
      fixed (int* p = (int*)&pt.x)   // CS0254  
      // try the following line instead  
      // fixed (uint* p = &pt.x)  
      {  
         SquarePtrParam ((int*)p);  
      }  
   }  
}  
```
