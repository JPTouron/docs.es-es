# <a name="compiler-error-cs0748"></a><span data-ttu-id="b73d5-102">Error del compilador CS0748</span><span class="sxs-lookup"><span data-stu-id="b73d5-102">Compiler Error CS0748</span></span>

<span data-ttu-id="b73d5-104">Si una expresión lambda tiene varios parámetros de entrada, algunos parámetros no pueden usar tipos implícitos mientras que otros usan tipos explícitos.</span><span class="sxs-lookup"><span data-stu-id="b73d5-104">If a lambda expression has multiple input parameters, some parameters cannot use implicit typing while others use explicit typing.</span></span>
