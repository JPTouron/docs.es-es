## <a name="example"></a><span data-ttu-id="88da7-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="88da7-109">Example</span></span>  

 <span data-ttu-id="88da7-110">El ejemplo siguiente genera la advertencia CS0688.</span><span class="sxs-lookup"><span data-stu-id="88da7-110">The following sample generates CS0688.</span></span> <span data-ttu-id="88da7-111">Para resolver la advertencia sin modificar la clase base, quite el atributo de seguridad del método de reemplazo.</span><span class="sxs-lookup"><span data-stu-id="88da7-111">To resolve the warning without modifying the base class, remove the security attribute from the overriding method.</span></span> <span data-ttu-id="88da7-112">No solucionará el problema de seguridad.</span><span class="sxs-lookup"><span data-stu-id="88da7-112">This will not solve the security problem.</span></span>  
  
```csharp  
// CS0688.cs  
// compile with: /W:1  
using System;  
using System.Security.Permissions;  
  
class Base
{  
    //Uncomment the following line to close the security hole  
    //[FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")]  
    public virtual void DoScaryFileStuff()  
    {  
    }  
}  
  
class Derived: Base  
{  
    [FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")] // CS0688  
    public override void DoScaryFileStuff()  
    {  
    }  
    static void Main()  
    {  
    }  
}  
```
