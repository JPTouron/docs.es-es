## <a name="example"></a><span data-ttu-id="d2375-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="d2375-108">Example</span></span>  

 <span data-ttu-id="d2375-109">El ejemplo siguiente genera la advertencia CS0411:</span><span class="sxs-lookup"><span data-stu-id="d2375-109">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
