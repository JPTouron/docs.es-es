# <a name="compiler-error-cs0156"></a><span data-ttu-id="a8a9f-103">Error del compilador CS0156</span><span class="sxs-lookup"><span data-stu-id="a8a9f-103">Compiler Error CS0156</span></span>
<span data-ttu-id="a8a9f-104">No se permite una instrucción throw sin argumentos en una cláusula finally anidada en la cláusula catch más cercana</span><span class="sxs-lookup"><span data-stu-id="a8a9f-104">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>  
  
 <span data-ttu-id="a8a9f-105">Una instrucción [throw](../language-reference/keywords/throw.md) sin parámetros solo puede aparecer en una cláusula **catch** que no tome parámetros.</span><span class="sxs-lookup"><span data-stu-id="a8a9f-105">A [throw](../language-reference/keywords/throw.md) statement with no parameters can only appear in a **catch** clause that takes no parameters.</span></span>  
  
 <span data-ttu-id="a8a9f-106">Para obtener más información, vea [excepciones y control](../programming-guide/exceptions/index.md)de excepciones.</span><span class="sxs-lookup"><span data-stu-id="a8a9f-106">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="a8a9f-107">El ejemplo siguiente genera la advertencia CS0156:</span><span class="sxs-lookup"><span data-stu-id="a8a9f-107">The following sample generates CS0156:</span></span>  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
