 <span data-ttu-id="654f4-105">En una instrucción de asignación, el valor del lado derecho se asignó al lado izquierdo.</span><span class="sxs-lookup"><span data-stu-id="654f4-105">In an assignment statement, the value of the right-hand side is assigned to the left-hand side.</span></span> <span data-ttu-id="654f4-106">El lado izquierdo debe ser una variable, una propiedad o un indexador.</span><span class="sxs-lookup"><span data-stu-id="654f4-106">The left-hand side must be a variable, property, or indexer.</span></span>

## <a name="example"></a><span data-ttu-id="654f4-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="654f4-109">Example</span></span>  

 <span data-ttu-id="654f4-110">El ejemplo siguiente genera la advertencia CS0131.</span><span class="sxs-lookup"><span data-stu-id="654f4-110">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="654f4-111">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="654f4-111">Example</span></span>  

 <span data-ttu-id="654f4-112">Este error también puede producirse si intenta realizar operaciones aritméticas en el lado izquierdo de un operador de asignación, como en el ejemplo siguiente.</span><span class="sxs-lookup"><span data-stu-id="654f4-112">This error can also occur if you attempt to perform arithmetic operations on the left hand side of an assignment operator, as in the following example.</span></span>  
  
```csharp  
// CS0131b.cs  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)  
            return 0;  
        return 1;  
    }  
}  
```
