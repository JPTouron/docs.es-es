## <a name="example"></a><span data-ttu-id="4d8c0-105">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="4d8c0-105">Example</span></span>

<span data-ttu-id="4d8c0-106">El ejemplo siguiente genera la advertencia CS0619:</span><span class="sxs-lookup"><span data-stu-id="4d8c0-106">The following sample generates CS0619:</span></span>  
  
```csharp
using System;

public class C
{
    [Obsolete("Use NewMethod instead", true)] // generates an error on use
    public static void OldMethod()
    {
    }

    // this is the method you should be using
    public static void NewMethod()
    {
    }  
}

class MyClass
{
   public static void Main()
   {
      C.OldMethod();   // CS0619
   }
}
```
