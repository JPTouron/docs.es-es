 <span data-ttu-id="3334c-105">Una declaración explícita de la [interfaz](../language-reference/keywords/interface.md) solo se puede declarar en una [clase](../language-reference/keywords/class.md) o una [estructura](../language-reference/builtin-types/struct.md).</span><span class="sxs-lookup"><span data-stu-id="3334c-105">An explicit [interface](../language-reference/keywords/interface.md) declaration was found outside a [class](../language-reference/keywords/class.md) or [struct](../language-reference/builtin-types/struct.md).</span></span>  
  
 <span data-ttu-id="3334c-106">El ejemplo siguiente genera la advertencia CS0541:</span><span class="sxs-lookup"><span data-stu-id="3334c-106">The following sample generates CS0541:</span></span>  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }

   {  
      void IFace.F();   // CS0541  
   }  
}  
```
