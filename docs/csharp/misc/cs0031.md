# <a name="compiler-error-cs0031"></a><span data-ttu-id="4fc20-103">Error del compilador CS0031</span><span class="sxs-lookup"><span data-stu-id="4fc20-103">Compiler Error CS0031</span></span>

<span data-ttu-id="4fc20-104">El valor constante 'valor' no se puede convertir en un 'tipo'.</span><span class="sxs-lookup"><span data-stu-id="4fc20-104">Constant value 'value' cannot be converted to a 'type'.</span></span>

<span data-ttu-id="4fc20-105">Se intentó asignar un valor a una variable cuyo tipo no puede almacenar el valor.</span><span class="sxs-lookup"><span data-stu-id="4fc20-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="4fc20-106">Para obtener más información, vea [Tipos](../programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="4fc20-106">For more information, see [Types](../programming-guide/types/index.md).</span></span>

<span data-ttu-id="4fc20-107">El ejemplo siguiente genera la advertencia CS0031 en contextos checked y unchecked:</span><span class="sxs-lookup"><span data-stu-id="4fc20-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp
// CS0031.cs
namespace CS0031
{
    public class Program
    {
        public static void Main()
        {
            int num = (int)2147483648M; //CS0031
            // Try using a larger numeric type instead.
            // long num = (long)2147483648M; //CS0031

            const decimal d = -10M; // Decimal literal
            unchecked
            {
                const byte b = (byte)d; // CS0031
                // For small values try using a signed byte instead.
                // const sbyte b = (sbyte)d;
            }
        }
    }
}
```
