# <a name="compiler-error-cs1650"></a><span data-ttu-id="c5904-103">Error del compilador CS1650</span><span class="sxs-lookup"><span data-stu-id="c5904-103">Compiler Error CS1650</span></span>

<span data-ttu-id="c5904-104">Los campos del campo de solo lectura estático 'identificador' no se pueden asignar (excepto en un constructor estático o un inicializador de variables).</span><span class="sxs-lookup"><span data-stu-id="c5904-104">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="c5904-105">Este error se produce cuando se intenta modificar un miembro de un campo que es estático y de solo lectura que no está permitida su modificación.</span><span class="sxs-lookup"><span data-stu-id="c5904-105">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="c5904-106">Para resolver este error, limite las asignaciones a campos de solo lectura para el constructor o inicializador de variable o quite la palabra clave `readonly` de la declaración del campo.</span><span class="sxs-lookup"><span data-stu-id="c5904-106">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
