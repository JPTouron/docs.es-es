# <a name="compiler-error-cs0175"></a>Error del compilador CS0175
El uso de la palabra clave 'base' no es válido en este contexto  
  
 La palabra clave [base](../language-reference/keywords/base.md) debe usarse para especificar un miembro determinado de la clase base. Para obtener más información, vea [Constructores](../programming-guide/classes-and-structs/constructors.md).  
  
 El ejemplo siguiente genera la advertencia CS0175:  
  
```csharp  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
