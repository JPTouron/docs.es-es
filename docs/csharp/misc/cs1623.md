# <a name="compiler-error-cs1623"></a><span data-ttu-id="9b6f4-103">Error del compilador CS1623</span><span class="sxs-lookup"><span data-stu-id="9b6f4-103">Compiler Error CS1623</span></span>
<span data-ttu-id="9b6f4-104">Los iteradores no pueden tener parámetros Ref o out</span><span class="sxs-lookup"><span data-stu-id="9b6f4-104">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="9b6f4-105">Este error se produce si un método iterador toma un `in` `ref` parámetro, o `out` .</span><span class="sxs-lookup"><span data-stu-id="9b6f4-105">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="9b6f4-106">Para evitar este error, quite la `in` `ref` `out` palabra clave, o de la firma del método.</span><span class="sxs-lookup"><span data-stu-id="9b6f4-106">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9b6f4-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="9b6f4-107">Example</span></span>  
 <span data-ttu-id="9b6f4-108">El ejemplo siguiente genera la advertencia CS1623:</span><span class="sxs-lookup"><span data-stu-id="9b6f4-108">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
