# <a name="compiler-error-cs1504"></a>Error del compilador CS1504
No se pudo abrir el archivo de código fuente 'file' ('reason')  
  
 El compilador no pudo abrir o leer un archivo de origen. Es posible que el archivo esté bloqueado por otra aplicación o puede que haya otro problema de sistema operativo. El mensaje incluye el motivo por el que el compilador no pudo abrir o leer el archivo del sistema operativo.
