# <a name="compiler-error-cs0554"></a><span data-ttu-id="4fca4-103">Error del compilador CS0554</span><span class="sxs-lookup"><span data-stu-id="4fca4-103">Compiler Error CS0554</span></span>

<span data-ttu-id="4fca4-104">'rutina de conversión': conversión definida por el usuario a clase derivada o desde ella</span><span class="sxs-lookup"><span data-stu-id="4fca4-104">'conversion routine' : user defined conversion to/from derived class</span></span>  
  
 <span data-ttu-id="4fca4-105">No se permiten las conversiones definidas por el usuario a valores de una clase derivada; no es necesario este tipo de operador.</span><span class="sxs-lookup"><span data-stu-id="4fca4-105">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="4fca4-106">Consulte el capítulo 6 en la especificación del lenguaje C# para obtener más información sobre las conversiones definidas por el usuario.</span><span class="sxs-lookup"><span data-stu-id="4fca4-106">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>  
  
 <span data-ttu-id="4fca4-107">El ejemplo siguiente genera la advertencia CS0554:</span><span class="sxs-lookup"><span data-stu-id="4fca4-107">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
