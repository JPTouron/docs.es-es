## <a name="example"></a><span data-ttu-id="e258f-112">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="e258f-112">Example</span></span>

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="e258f-114">Corrija el error mediante `this.num` para hacer referencia al campo de clase.</span><span class="sxs-lookup"><span data-stu-id="e258f-114">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
