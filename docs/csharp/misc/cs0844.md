## <a name="example"></a><span data-ttu-id="ce2b3-112">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="ce2b3-112">Example</span></span>

    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844

```csharp

    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
