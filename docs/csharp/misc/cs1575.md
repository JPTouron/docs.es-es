# <a name="compiler-error-cs1575"></a><span data-ttu-id="0a69e-103">Error del compilador CS1575</span><span class="sxs-lookup"><span data-stu-id="0a69e-103">Compiler Error CS1575</span></span>

<span data-ttu-id="0a69e-104">Una expresión stackalloc necesita []; después del tipo.</span><span class="sxs-lookup"><span data-stu-id="0a69e-104">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="0a69e-105">El tamaño de la asignación solicitada, con [stackalloc](../language-reference/operators/stackalloc.md), debe especificarse entre corchetes.</span><span class="sxs-lookup"><span data-stu-id="0a69e-105">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="0a69e-106">El ejemplo siguiente genera la advertencia CS1575:</span><span class="sxs-lookup"><span data-stu-id="0a69e-106">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
