 <span data-ttu-id="0c08d-105">Se usó la palabra clave [base](../language-reference/keywords/base.md) fuera de un método, una propiedad o un constructor.</span><span class="sxs-lookup"><span data-stu-id="0c08d-105">The [base](../language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>

```csharp  
{  
    private String xx = base.ToString();   // CS1512

    //    xx = base.ToString();  
    // }

    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
