# <a name="compiler-error-cs1512"></a>Error del compilador CS1512

La palabra clave 'base' no está disponible en el contexto actual  
  
 Se usó la palabra clave [base](../language-reference/keywords/base.md) fuera de un método, una propiedad o un constructor.  
  
 El siguiente ejemplo genera el error CS1512:  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
