# <a name="compiler-error-cs1512"></a><span data-ttu-id="30c48-103">Error del compilador CS1512</span><span class="sxs-lookup"><span data-stu-id="30c48-103">Compiler Error CS1512</span></span>

<span data-ttu-id="30c48-104">La palabra clave 'base' no está disponible en el contexto actual</span><span class="sxs-lookup"><span data-stu-id="30c48-104">Keyword 'base' is not available in the current context</span></span>  
  
 <span data-ttu-id="30c48-105">Se usó la palabra clave [base](../language-reference/keywords/base.md) fuera de un método, una propiedad o un constructor.</span><span class="sxs-lookup"><span data-stu-id="30c48-105">The [base](../language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="30c48-106">El siguiente ejemplo genera el error CS1512:</span><span class="sxs-lookup"><span data-stu-id="30c48-106">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
