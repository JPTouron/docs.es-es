# <a name="compiler-error-cs0174"></a>Error del compilador CS0174
Clase base requerida para una referencia 'base'  
  
 Este error solo se produce cuando el tiempo de ejecución de .NET compila el código fuente de la `System.Object` clase, que es la única clase que no tiene clase base.
