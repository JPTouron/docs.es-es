<span data-ttu-id="e3bc0-104">'miembro' en la declaración explícita de la interfaz no es un miembro de interfaz.</span><span class="sxs-lookup"><span data-stu-id="e3bc0-104">'member' in explicit interface declaration is not a member of interface</span></span>  
  
 <span data-ttu-id="e3bc0-105">Se intentó declarar explícitamente un miembro de [interfaz](../language-reference/keywords/interface.md) que no existe.</span><span class="sxs-lookup"><span data-stu-id="e3bc0-105">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="e3bc0-106">Debe eliminar la declaración o cambiarla para que haga referencia a un miembro de interfaz válido.</span><span class="sxs-lookup"><span data-stu-id="e3bc0-106">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>  
  
 <span data-ttu-id="e3bc0-107">El ejemplo siguiente genera la advertencia CS0539:</span><span class="sxs-lookup"><span data-stu-id="e3bc0-107">The following sample generates CS0539:</span></span>  
  
```csharp

   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
