<span data-ttu-id="acc71-103">.NET Framework 4 presenta cinco nuevos tipos de colección que están especialmente diseñadas para admitir operaciones multiproceso de agregar y quitar.</span><span class="sxs-lookup"><span data-stu-id="acc71-103">The .NET Framework 4 introduces five new collection types that are specially designed to support multi-threaded add and remove operations.</span></span> <span data-ttu-id="acc71-104">Para obtener seguridad para los subprocesos, estos nuevos tipos usan diversas clases de mecanismos de sincronización eficientes con bloqueo y sin bloqueo.</span><span class="sxs-lookup"><span data-stu-id="acc71-104">To achieve thread-safety, these new types use various kinds of efficient locking and lock-free synchronization mechanisms.</span></span> <span data-ttu-id="acc71-105">La sincronización agrega sobrecarga a las operaciones.</span><span class="sxs-lookup"><span data-stu-id="acc71-105">Synchronization adds overhead to an operation.</span></span> <span data-ttu-id="acc71-106">La cantidad de sobrecarga depende del tipo de sincronización que se use, el tipo de operaciones que se realicen y otros factores, como el número de subprocesos que intentan obtener acceso simultáneamente a la colección.</span><span class="sxs-lookup"><span data-stu-id="acc71-106">The amount of overhead depends on the kind of synchronization that is used, the kind of operations that are performed, and other factors such as the number of threads that are trying to concurrently access the collection.</span></span>
