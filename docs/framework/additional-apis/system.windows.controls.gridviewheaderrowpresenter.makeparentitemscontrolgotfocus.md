# <a name="makeparentitemscontrolgotfocus-method"></a>Método MakeParentItemsControlGotFocus

Proporciona el foco al control primario del elemento. Si el control primario es un <xref:System.Windows.Controls.ListBox> , proporciona el foco al elemento al que se ha accedido más recientemente en <xref:System.Windows.Controls.ListBox> .

```csharp
internal void MakeParentItemsControlGotFocus()
```

> [!WARNING]
> Este método es interno y no está diseñado para usarse directamente en el código.
>
> Microsoft no admite el uso de este método en una aplicación de producción bajo ninguna circunstancia.
