# <a name="connectstreamconnection-property"></a>Propiedad ConnectStream. Connection

Obtiene la conexión.

## <a name="syntax"></a>Sintaxis

```csharp
internal Connection Connection { get; }
```

## <a name="property-value"></a>Valor de propiedad

`System.Net.Connection`

Objeto de conexión.

## <a name="remarks"></a>Comentarios

> [!WARNING]
> La `ConnectStream.Connection` propiedad es interna y no está diseñada para usarse directamente en el código.
>
> Microsoft no admite el uso de esta propiedad en una aplicación de producción bajo ninguna circunstancia.
