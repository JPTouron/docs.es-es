# <a name="connectionm_writelist-field"></a>Campo Connection. m \_ WriteList

`Connection.m_WriteList`es una <xref:System.Collections.ArrayList> de los <xref:System.Net.HttpWebRequest> objetos que se ponen en cola para enviarse a través de http.

## <a name="syntax"></a>Sintaxis
  
```csharp  
private ArrayList m_WriteList
```

> [!WARNING]
> El `Connection.m_WriteList` campo es privado y no está diseñado para usarse directamente en el código.
>
> Microsoft no admite el uso de este campo en una aplicación de producción en cualquier circunstancia.

## <a name="requirements"></a>Requisitos

**Espacio de nombres:** <xref:System.Net>

**Ensamblado:** Sistema (en System.dll)

**.NET Framework versiones:** Disponible desde 2,0.
