### <a name="the-connectionstrings-section"></a><span data-ttu-id="a7bb1-112">Sección connectionStrings</span><span class="sxs-lookup"><span data-stu-id="a7bb1-112">The connectionStrings Section</span></span>

```xml  
    <connectionStrings>  
      <clear />

    </connectionStrings>  
  </configuration>  
```
