# <a name="supported-and-unsupported-linq-methods-linq-to-entities"></a>Métodos de LINQ compatibles y no compatibles (LINQ to Entities)
En esta sección se proporciona información sobre los operadores de consulta estándar de Language-Integrated Query (LINQ) que se admiten o no se admiten en consultas de LINQ to Entities. Muchos de los operadores de consulta estándar de LINQ tienen una versión sobrecargada que acepta un argumento entero. El argumento entero corresponde a un índice basado en cero en la secuencia en la que se está operando, <xref:System.Collections.Generic.IEqualityComparer%601> o <xref:System.Collections.Generic.IComparer%601> . A menos que se especifique lo contrario, estas versiones sobrecargadas de los operadores de consulta estándar de LINQ no se admiten y, si intenta utilizarlos, se iniciará una excepción.

## <a name="type-methods"></a>Métodos de tipos  
 Los operadores de consulta estándar de LINQ que se ocupan de la conversión y prueba de tipos de CLR se admiten en el Entity Framework. LINQ to Entities admite solo los tipos de CLR que se asignan a los tipos del modelo conceptual. Para obtener una lista de los tipos de modelos conceptuales, vea [tipos de modelos conceptuales (CSDL)](/ef/ef6/modeling/designer/advanced/edmx/csdl-spec#conceptual-model-types-csdl). La tabla siguiente enumera los métodos de tipos admitidos y no admitidos.  
  
|Método|Soporte técnico|Firma de funciones de Visual Basic|Firma de métodos de C#|  
|------------|-------------|-------------------------------------|--------------------------|  
|<xref:System.Linq.Queryable.Cast%2A>|Se admite para los tipos primitivos de EDM|`Function Cast(Of TResult) ( _ source As IQueryable _ ) As IQueryable(Of TResult)`|`IQueryable<TResult> Cast<TResult>( this IQueryable source )`|  
|<xref:System.Linq.Queryable.OfType%2A>|Se admite para <xref:System.Data.Metadata.Edm.EntityType>|`Function OfType(Of TResult) ( _ source As IQueryable _ ) As IQueryable(Of TResult)`|`IQueryable<TResult> OfType<TResult>( this IQueryable source )`|
