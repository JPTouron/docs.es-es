# <a name="datetimepicker-control-windows-forms"></a>DateTimePicker (Control, formularios Windows Forms)
El `DateTimePicker` control Windows Forms permite al usuario seleccionar un único elemento de una lista de fechas u horas. Cuando se usa para representar una fecha, aparece en dos partes: una lista desplegable con una fecha representada en el texto y una cuadrícula que aparece al hacer clic en la flecha hacia abajo situada junto a la lista.
