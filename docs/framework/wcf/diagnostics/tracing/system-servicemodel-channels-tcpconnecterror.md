# <a name="systemservicemodelchannelstcpconnecterror"></a>System.ServiceModel.Channels.TcpConnectError
Error en la operación de conexión TCP.  
  
## <a name="description"></a>Descripción  
 Este seguimiento de traza de nivel de advertencia indica un error en la conexión a un extremo TCP. Esto podría suceder si el punto de conexión remoto no responde en una dirección IP y puerto determinados. Puede omitirse este seguimiento de traza si se realizan correctamente otros intentos posteriores de conexión con otras direcciones IP válidas (como direcciones IPv4 o IPv6, u otras direcciones IP que representan un nombre de host concreto). La excepción de este seguimiento de traza puede revelar información adicional acerca del error.
