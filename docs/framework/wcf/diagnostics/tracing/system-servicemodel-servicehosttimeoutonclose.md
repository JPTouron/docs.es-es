# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Descripción  
 Se agotó el tiempo de espera de la operación de cierre de ServiceHost. Esto indica que algunos canales con sesión no se cerraron correctamente en el cliente y podrían haberse anulado. Modifique el tiempo predeterminado para ajustar el tiempo de espera de cierre.
