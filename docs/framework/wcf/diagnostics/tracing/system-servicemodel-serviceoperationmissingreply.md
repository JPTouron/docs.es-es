# <a name="systemservicemodelserviceoperationmissingreply"></a>System.ServiceModel.ServiceOperationMissingReply
System.ServiceModel.ServiceOperationMissingReply  
  
## <a name="description"></a>Descripción  
 La operación de solicitud/respuesta no tiene ningún mensaje de respuesta.  
  
 Ésta es una operación bidireccional; sin embargo, no hay ninguna respuesta para enviar al cliente. Puede utilizar este seguimiento para depurar durante el desarrollo.
