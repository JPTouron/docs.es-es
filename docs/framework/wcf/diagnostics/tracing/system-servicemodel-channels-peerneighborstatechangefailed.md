# <a name="systemservicemodelchannelspeerneighborstatechangefailed"></a>System.ServiceModel.Channels.PeerNeighborStateChangeFailed
El estado del vecino estaba dañado.  
  
## <a name="description"></a>Descripción  
 Este seguimiento se produce después de realizar un intento fallido de cambiar el estado de un vecino. Esto indica un error interno inesperado o que la memoria está dañada.
