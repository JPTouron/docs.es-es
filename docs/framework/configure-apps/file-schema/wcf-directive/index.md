# <a name="wcf-directive-syntax"></a>Sintaxis de directivas WCF
Especifica la configuración usada por las directivas de Windows Communication Foundation (WCF) en los archivos. SVC para dirigir los compiladores. Cada directiva puede contener uno o varios atributos (emparejados con valores), que son específicos de esa directiva. WCF solo tiene la Directiva de [ \@ ServiceHost](servicehost.md) .  
  
## <a name="in-this-section"></a>En esta sección  
 [@ServiceHost](servicehost.md)  
 Define atributos específicos de página usados por el compilador de .svc. Solo se puede incluir en archivos .svc.
