# <a name="igchostcontrol-interface"></a>IGCHostControl (Interfaz)
Proporciona un método que permite al recolector de elementos no utilizados solicitar al host que cambie los límites de la memoria virtual.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Método RequestVirtualMemLimit](igchostcontrol-requestvirtualmemlimit-method.md)|Solicita al host que cambie los límites de la memoria virtual.|
