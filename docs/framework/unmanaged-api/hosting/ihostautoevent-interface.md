# <a name="ihostautoevent-interface"></a>IHostAutoEvent (Interfaz)
Proporciona una representación de la implementación del host de un evento de restablecimiento automático.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Método Set](ihostautoevent-set-method.md)|Establece la `IHostAutoEvent` instancia actual en un estado señalado.|  
|[Wait (Método)](ihostautoevent-wait-method.md)|Hace que la `IHostAutoEvent` instancia actual espere hasta que el evento sea propiedad o transcurra un período de tiempo especificado.|
